06 Example: Validate User Input

Purpose

Prove that you can write a Python program that validates user input and uses that input in calculations.

Problem Statement

While entering input for a program, users occasionally make mistakes and enter invalid or incorrect input. A good program recovers from user mistakes, informs users about mistakes in clear polite language, and gives users an opportunity to correct their mistakes.

Helpful Documentation

The code example in this lesson's prepare content shows how to require a user to enter a non-negative integer between a minimum and maximum value.

Assignment

The video for the previous lesson showed how to write a program that computes the surface area of a right circular cone. Change the program so that it validates all both of the numbers that the user enters while using the program. Assume that the user will enter integers and not floating point numbers (numbers with a decimal point). The minimum and maximum for the two numbers are as follows:

Quantity Minimum Maximum
radius 2 500
height 3 1000

Sample Run

> python example.py
This program computes and outputs the
surface area of a right circular cone.

Enter the integer radius of a cone: radius
'radius' is not a positive integer. Please enter a positive integer.
Enter the integer radius of a cone: 1
1 is too small. Please enter a positive integer between 2 and 500.
Enter the integer radius of a cone: 650
650 is too large. Please enter a positive integer between 2 and 500.
Enter the integer radius of a cone: 14

Enter the integer height of a cone: height
'height' is not a positive integer. Please enter a positive integer.
Enter the integer height of a cone: 1
1 is too small. Please enter a positive integer between 3 and 1000.
Enter the integer height of a cone: 9000
9000 is too large. Please enter a positive integer between 3 and 1000.
Enter the integer height of a cone: 20

The surface area is 1689.5

Testing Procedure

Verify that your program works correctly by following each step in this testing procedure:

  1. Run your program and enter the inputs shown in the Sample Run section above. Ensure that your program's output matches the sample run output.
