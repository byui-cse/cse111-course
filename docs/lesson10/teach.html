Assignment

Write a Python program that uses pandas and includes several functions to answer these two questions about the water usage data from the City of Rexburg:

  1. Is the total amount of water used in the City of Rexburg increasing each year?
  2. Is the average amount of water used per dwelling increasing each year?

Your program should compute actual numbers from the water meter data that answer the two questions above. Your program should print those numbers formatted nicely as shown in the sample run below and should draw at least two plots to help a user visualize the data that is printed out.

Note: If you examine closely the water data, you will see that some water meters serve both a business and one or more apartments. When analyzing data to answer the second question, your program should exclude the water usage data for such meters because there is no way to determine how much water went to the business and how much went to the attached apartment. In fact, when answering the second question, your program should use only the data for apartment complexes, residences, town homes, and trailer courts.

Core Requirements

  1. Your program must be split into at least three functions:
    1. main
    2. A function that shows a plot to answer the first question in this assigment.
    3. A function that shows a plot to answer the second question in this assigment.
  2. Your program must call pd.read_csv to read the water.csv file into a DataFrame and must add to the DataFrame a new column named "year" that contains only the year part of the "readDate" column.
  3. Your program must compute the sum of the water usage grouped by year and must display a vertical bar chart that shows total water usage in Rexburg by year.

Stretch Challenges

If your team finishes the core requirements in less than an hour, complete one or more of these stretch challenges. Note that the stretch challenges are optional.

  1. Add code to filter, group, and aggregate the water usage data and to display a vertical bar chart that answers the second question of this assignment.

Testing Procedure

Verify that your program works correctly by following each step in this testing procedure:

  1. Run your program and verify that it prints the correct results, rounded and formatted as shown in the sample run above.
  2. Examine the two vertical bar charts displayed by your program and verify that the two charts show the same values as the printed results.
