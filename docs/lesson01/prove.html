Digital images play an important role in our society today. Because of this, many programs have been written to analyze and manipulate them. One way that images can be stored and accessed is as a large set of individual pixels, each containing a red, green, and blue value from 0-255.

For this project, you'll work with an image of your choosing. Using Python, you will open an input image file and loop through each of the pixels, one by one. Depending on the value of these pixels, you'll need to make certain decisions to potentially change a pixel's value, and save it out as a new, modified output image file.

For this project, we will use an pre-built library (Python programs that others have written that we can use!) to help us open the images. Please be aware that a library like this is very powerful and can help you do much more than we are using it for, but for this lesson, we simply want the ability to iterate through the pixels one at a time.

Milestone Requirements

To ensure that you are prepared to complete the overall assignment by the end of the next lesson, for this lesson you need to complete the following:

  1. Have the necessary image libraries loaded.

  2. Obtain a picture of your choosing and put that file in the same directory as your Python program.

  3. Have your program open the image file and iterate through the pixels one by one.

  4. Add 50 to the green value of each pixel.

  5. Save out a new image (with the higher green values).

  6. You can then use a standard image program to open the new image and see the results.
