01 Prepare Your Development Environment

Each student in CSE 111 must install and use three pieces of free software:

  • The Python Programming Language version 3.8
  • Visual Studio Code (also known as VS Code)
  • The Python extension in VS Code

If you recently completed CSE 110, it is possible that you already have all three pieces of software installed on your computer. Otherwise, follow these steps to install the software:

  1. Download and install the latest stable version of Python 3.8. Some students report that Python 3.9 won't work for some of the assignments in this course. As of November 2020, the latest stable version of Python 3.8 is Python 3.8.6.
    1. Open the Python downloads page in your browser.
    2. Download and save the Python 3.8 installer for your operating system.
    3. Start installing Python by double clicking the file that you downloaded.

    4. On the first screen of the installer, be certain to check both boxes: "Install launcher..." and "Add Python 3.8 to PATH" as shown in this screen shot:

    5. Click the "Install Now" link and Python will be installed on your computer.
  2. Download and install VS Code.
    1. Open the Visual Studio Code download page in your browser.
    2. Download VS Code for your operating system.
    3. Install VS Code.
    4. Accept all the default settings.
  3. Create a folder on your computer where you will store all the programs that you write for CSE 111 and download and save this sample Python file into the folder that you created.
  4. Install the Python extension in VS Code.
    1. Run VS Code on your computer.
    2. In VS Code, open the sample Python file that you downloaded.
    3. In the lower right corner of VS Code, a small dialog will open that says, "The 'Python' extension is recommended for this file type."
    4. Click the "Install" button in that dialog.
  5. Execute the sample Python program.
    1. Execute the sample Python file that you downloaded by clicking the run icon in the upper right corner of VS Code. The run icon is a green triangle.
    2. The sample file should run without errors, and its output should be similar to this: 
      Hello! What is your name? Amelia
