01 Example: Review Python

Purpose

Review how to write a Python program that gets input from a user, uses variables, performs arithmetic using the math module, and outputs results.

Problem Statement

The surface area of a right circular cone can be computed with this formula:

A = π r
r +
√
r2 + h2
  • A is the surface area of the cone
  • r is the radius of the cone
  • h is the height of the cone

Assignment

Write a Python program named cone_surface_area.py that reads from the keyboard the radius and height of a right circular cone and computes and outputs the surface area of that cone.

Helpful Documentation

The Python math module contains mathematical constants and functions including math.pi and math.sqrt.
