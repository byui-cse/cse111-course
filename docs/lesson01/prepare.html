Prepare Your Development Environment

Each student in CSE 111 must install and use three pieces of free software: 1) The Python Programming Language (version 3.8 or later), 2) Visual Studio Code (also known as VS Code) and 3) the Python extension in VS Code. If you recently completed CSE 110, it is possible that you already have all pieces of software installed on your computer. Do the following:

  1. Download and install the latest stable version of Python
    1. Open https://www.python.org/downloads/release in your browser.
    2. Download the latest stable version of Python for your operating system.
    3. Install Python.
    4. Accept all the default settings.
  2. Create a folder on your computer where you will store all the programs that you write for CSE 111.
  3. Download and install VS Code
    1. Open https://code.visualstudio.com/download in your browser.
    2. Download VS Code for your operating system.
    3. Install VS Code.
    4. Accept all the default settings.
  4. Install the Python extension in VS Code
    1. Run VS Code on your computer.
    2. Download and save this Python file.
    3. Open in VS Code the file that you downloaded.
    4. In the lower right corner of VS Code, a small dialog will open that says, "The 'Python' extension is recommended for this file type."
    5. Click the "Install" button in that dialog.
  5. Execute a sample Python program
    1. Execute the Python file that you downloaded by first opening the file in VS Code and then clicking the run icon in the upper right corner of VS Code. The run icon is a green triangle.
    2. The file should run without errors and its output should be: 
      Hello! What is your name? <your name>

Review Python

Here is a list of the Python programming concepts and topics from CSE 110 that you should review this week.

  1. We use the input() function to prompt a user for input.
  2. Python has several data types including str, bool, int, float, complex, tuple, list, set, dict, bytearray, function, and type
  3. It is possible to convert between many of the data types. For example, to convert from any data type to a string, we use the str() function. To convert from a string to an integer, we use the int() function.
  4. In Python, we assign a value to a variable by using the assignment operator which is the equals symbol (=).
  5. Python has many arithmetic operators including power (**), negation (-), multiplication (*), division (/), floor division (//), modulo (%), addition (+), and subtraction (-).
  6. When we write an arithmetic expression that contains more than one operator, the operators are executed according to the order of operations. This table shows the order of operations for the arithmetic operators.
    ( ) highest precedence
    **
    - (negation)
    * / // %
    + -
    = lowest precedence
  7. We use the print() function to display output to a user.
