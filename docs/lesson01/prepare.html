CSE 111 | Programming with Functions

01 Prepare: Development Environment

Each student in CSE 111 must install and use three pieces of free software: 1) The Python Programming Language (version 3.8), 2) Visual Studio Code (also known as VS Code) and 2) the python extension in VS Code. If you recently completed CSE 110, it is possible that you already have both pieces of software already installed on your computer.

  1. Download and install Python 3.8
    1. Open https://www.python.org/downloads/release/python-380/ in your browser.
    2. Download Python 3.8 for your operating system.
    3. Install Python.
    4. Accept all the default settings.
  2. Download and install VS Code
    1. Open https://code.visualstudio.com/download in your browser.
    2. Download VS Code for your operating system.
    3. Install VS Code.
    4. Accept all the default settings.
  3. Install the python extension in VS Code
    1. Run VS Code on your computer.
    2. Open a python file (such as this one) in VS Code.
    3. In the lower right corner of VS Code, a small dialog will open that says, "The 'Python' extension is recommended for this file type."
    4. Click the "Install" button.
  4. Create a folder on your computer where you will store all the programs that you write for CSE 111.
