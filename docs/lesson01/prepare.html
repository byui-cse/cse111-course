01 Prepare: Review Python

Here is a list of the Python programming concepts and topics from CSE 110 that you should review this week.

  1. We use the input() function to prompt a user for input.
  2. In Python, we assign a value to a variable by using the assignment operator which is the equals symbol (=).
  3. Python has several data types including str, bool, int, float, complex, tuple, list, set, dict, bytearray, function, and type
  4. It is possible to convert between many of the data types. For example, to convert from any data type to a string, we use the str() function. To convert from a string to an integer, we use the int() function.
  5. Python has many arithmetic operators including power (**), negation (-), multiplication (*), division (/), floor division (//), modulo (%), addition (+), and subtraction (-).

  6. When we write an arithmetic expression that contains more than one operator, the operators are executed according to the order of operations. This table shows the order of operations for the arithmetic operators.

    Operators Precedence
    ( ) highest precedence
    **
    - (negation)
    * / // %
    + -
    = lowest precedence
  7. We use the print() function to display output to a user.
