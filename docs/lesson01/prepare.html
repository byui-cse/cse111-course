01 Prepare: Review Python

Concepts

Here is a list of the Python programming concepts and topics from CSE 110 that you should review during this lesson.

  1. We use the input() function to prompt a user for input.
  2. In Python, we assign a value to a variable by using the assignment operator which is the equals symbol (=).
  3. Python has several data types including str, bool, int, float, complex, tuple, list, set, dict, function, and type. This program demonstrates many of the Python data types.
  4. It is possible to convert between many of the data types. For example, to convert from any data type to a string, we use the str() function. To convert from a string to an integer, we use the int() function, and to convert from a string to a float, we use the float() function. The int() and float() functions are especially useful to convert user input, which is always a string, from a string to a number. See the example program below.
  5. Python has many arithmetic operators including power (**), negation (-), multiplication (*), division (/), floor division (//), modulo (%), addition (+), and subtraction (-).

  6. When we write an arithmetic expression that contains more than one operator, the operators are executed according to the order of operations. This table shows the order of operations for the arithmetic operators.

    Operators Precedence
    ( ) highest precedence
    **
    - (negation)
    *   /   //   %
    +   -
    = lowest precedence

  7. We use the print() function to display output to a user.

    The following program gets input from the user, converts the input to two numbers, computes the length of a cable from the two numbers, rounds the length to two places after the decimal point, and prints the rounded length for the user to see.

    # This program computes the length of a cable given the distance
# it will span and the distance it will sag or dip in the middle.

# Get user input and convert it from
# strings to floating point numbers.
span = float(input("Distance the cable must span in meters: "))
dip = float(input("Distance the cable will sag in meters: "))

# Use the numbers to compute the cable length.
length = span + 8 * dip**2 / (3 * span)

# Round the length to two places after the decimal point.
length = round(length, 2)

# Print the cable length in the
# terminal window for the user to see.
print(f"Length of cable in meters: {length}")

    > python cable.py
Distance the cable must span in meters: 500
Distance the cable will sag or dip in meters: 18.5
Length of cable in meters: 501.83
  8. We can store several values in a Python tuple. Tuples are immutable, meaning after a tuple is created, it cannot be changed. Tuples are most useful for data such as geometric points that contain x and y parts.
  9. We can store many items in a Python list. Lists are mutable, meaning they can be changed. We can add an item to a list using insert and append. We can replace an item in a list and retrieve an item from a list using the index operators ([ and ]). We can remove an item from a list using del, pop, and remove. We can find an item in a list using index, and we can determine the number of items in a list using the len function.
  10. We can store many items in a Python set. Sets are mutable, meaning they can be changed. A set is different from a list. A list allows duplicate items to be stored in it, but a set doesn't allow duplicate items. Items in a set are unique.

  11. We can also store many items in a Python dictionary. Dictionaries are mutable, meaning they can be changed. In a dictionary, items are stored as key value pairs and we can easily write code to find a value by its key.

    The following example code creates one each of a tuple, list, set, and dictionary:

    # Get two numbers from the user.
x = float(input("Enter the x value of a point: "))
y = float(input("Enter the y value of a point: "))

# Create a tuple.
point = (x, y)

# Create a list.
samples = [2.5, -3.2, 2.1, 0.6, 2.1, 0.7]

# Create a set.
colors = {"red", "green", "blue"}

# Create a dictionary.
students = {
    "67-412-8350" : "Alan Benson",
    "76-240-1835" : "Madison Silverlake",
    "06-412-7583" : "Samyukta Daniels"
}
