01 Class Activity: Swing Time for a Pendulum

Instructions

With your class write a simple Python program.

Problem Statement

The time in seconds that a pendulum takes to swing back and forth once is given by this formula:

t = 2π √
n
9.81
  • where t is the time in seconds,
  • π is PI, the ratio of the circumference divided by the diameter of a circle (use math.pi), and
  • n is the length of the pendulum in meters.

Activity

Download and save this file: pendulum.py. Open the file in VS Code or another text editor and examine it. Notice that it contains a triple quoted string which acts as a comment for the entire program.

Write a program that prompts a user to enter the length of a pendulum in meters and then computes and prints the time in seconds that it takes for that pendulum to swing back and forth.

Helpful Documentation

The Python math module contains mathematical constants and functions including math.pi and math.sqrt().

Sample Run

> python pendulum.py
Length of pendulum (meters): 3
Time (seconds): 3.47

Sample Solution

Please work diligently with your class to complete this activity. After class is over, please compare your approach to the sample solution. Please do not look at the sample solution until you have either finished the program or diligently worked with your class.
