Problem Statement

Star Wars is a series of movies about fictional characters and events that lived and happened a long time ago in a galaxy far far away. Because the movies are popular, there are many fan websites that contain information about the movies. One such website is swapi.co swapi.co maintains a web service that contains lots of data gleaned from the movies and books about the fictional characters and places that are part of the Star Wars galaxy. During this activity, your team will write a Python program that uses the requests module to retrieve data from the swapi service.

Helpful Documentation

The Python requests module contains functions to send http requests for data to computer servers on the internet and to read the data that a server returns. Read about the requests module at requests.readthedocs.io

The Star Wars API (swapi) maintains lots of data about the ficitional Star Wars galaxy. Read about swapi and experiment with it at swapi.co/documentation

Instructions

Use the commodities.py program, that you downloaded during the checkpoint for this lesson, as a reference.

Write a Python program that reads the contents of the quotes.txt file into a list and then randomly chooses one of the quotes and prints it for the user to see. Include these two lines of code at the top of your program to import the functions in the random and textwrap modules:

import random
import textwrap

Core Requirements

  1. Your program must read the quotes from the quotes.txt file into a list and randomly choose one of the quotes from the list.
  2. Your program must wrap the words in the quote onto multiple lines if the quote is longer than 70 characters and then print the wrapped quote for the user to see.
  3. Your program must call the following functions:
    • open - to open the quotes.txt file.
    • file.readlines - to read all the lines in the file as a list
    • random.choice - to randomly choose one of the quotes
    • textwrap.wrap - to wrap the words of the chosen quote to 70 characters or fewer per line
    • print - to print the wrapped lines of the quote

Stretch Challenges

If your team finishes the core requirements in less than an hour, complete one or more of these stretch challenges. Note that the stretch challenges are optional.

  1. Add something or change something in your program that you think would make your program better, easier for the user, more elegant, or more fun. Be creative.
  2. Write comments above the lines of code that were difficult for you to understand.
  3. Change the call to the textwrap.wrap function to wrap the words in the quote to 50 characters instead of 70.
  4. Allow the user to enter a positive integer to choose how quotes are chosen and printed.
