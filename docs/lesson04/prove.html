Overview

Prove that you can write a Python program that calls third-party functions.

Problem Statement

The weather affects people's lives for good and bad every day. Because of the impact of the weather on people's lives, the United States Federal Government sponsors the U.S. National Weather Service which forecasts the weather throughout the entire United States every day. These weather forecasts are available for free as a web service on the internet. Write a Python program that

  1. allows a user to select a city from a list of cities
  2. retrieves weather forecasts for the next seven days and nights from the U.S. National Weather Service
  3. writes the forecasts to an HTML file that a user can view in a browser

During lesson 3, you wrote the part of this program that writes a weather forecast to an HTML file. During this assignment (lesson 4), you will add the code to prompt the user for their desired city and to retrieve a forecast from the U.S. Weather Service.

Helpful Documentation

The website latlong.net allows a user to enter the name and state of a city in the U.S. and then displays the latitude and longitude of that city.

The Python requests module contains functions to send http requests for data to a computer server on the internet and to read the data that the server returns.

The U.S. National Weather Service maintains a service that returns weather forecasts for cities within the United States. The service is documentated at www.weather.gov/documentation/services-web-api.

The Python weather_html module was written at BYU-Idaho and accepts U.S. National Weather Service forecasts and writes HTML to a file.

Assignment

Open your program from the prove assignment for last week and add the following to it:

  1. A dictionary with at least five U.S. cities and their corresponding latitude and longitude. A dictionary with this structure would work well:
    Keys (string) Values (tuple)
    "Dallas, Texas" (32.776665, -96.796989)
    "Rexburg, Idaho" (43.822893, -111.790988)
  2. Python code that allows a user to choose one of the cities in the dictionary.
  3. Python code that uses the requests module to retrieve the weather forecast for the city that the user chose.

You should keep the code that you wrote last week to generate an HTML report. You should connect the code from step 3 above with the code that you wrote last week to generate an HTML report.

Testing Procedure

Verify that your program works correctly by following each step in this testing procedure:

  1. Run your program, choose one of the cities, verify that your program wrote an HTML file to your computer's hard drive, and double click on the HTML file that you program produced. Double clicking the HTML file should open it in a browser. Look at the document in the browser and verify that it looks correct, including the correct city name and a correctly formatted five day, vertical forecast.
  2. Repeat the above step for each of the four other cities in your program.
  3. Repeat step 1 again for each of the five cities, but do it at a different time of the day. For example, if you first tested your program during the morning, then test it again in the afternoon or vice versa.
