Overview

Prove that you can write a Python program that calls built-in and standard functions.

Problem Statement

The weather affects people's lives for good and bad every day. Because of the impact of the weather on people's lives, the United States Federal Government sponsors the U.S. National Weather Service which forecasts the weather throughout the entire United States every day. These weather forecasts are available for free as a web service on the internet. Write a Python program that

  1. allows a user to select a city from a list of cities
  2. retrieves weather forecasts for the next seven days and nights from the U.S. National Weather Service
  3. writes the forecasts to an HTML file that a user can view in a browser

During lesson 3, you will write half of this Python program. This half will not retrieve the forecast from the U.S. Weather Service but will instead use two sets of test forecasts and will write those forecasts to an HTML file. During lesson 4, you will add the code to prompt the user for their desired city and to retrieve the forecasts from the U.S. Weather Service.

Helpful Documentation

The Python weather_html module was written at BYU-Idaho and accepts U.S. National Weather Service data and writes HTML to a file. To prepare for this assignment, read about weather_html module and its functions.

Assignment

Download both of these files: weather_html.py test_periods.py and save them into the same folder where you will save your Python program. Open the weather_html.py file and examine it. Notice that this file is the weather_html module and contains the four functions: create_report, add_row, add_period, write_report that are described in the module's documentation. Open the test_periods.py file and notice that it contains two lists: periods_that_start_with_a_day and periods_that_start_with_a_night. You will use these two lists to test your program during week 3.

Write a Python program that creates a weather forecast report in one of these three formats:

  • horizontal three day

Testing Procedure

Verify that your program works correctly by following each step in this testing procedure:

  1. Run your program and double click on the HTML file that you program produces. Double clicking the HTML file should open it in a browser. Look at the document in the browser and verify that it looks matches the corresponding document from the table above.
  2. Change your program to use the list periods_that_start_with_a_night and then run your program again. Double click on the HTML file and verify that the document changed and that it looks matches the corresponding document from the table above.
