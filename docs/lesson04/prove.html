CSE 111 | Programming with Functions

04 Prove Assignment: Calling Functions

Overview

Prove that you can write a Python program that calls functions and methods.

Problem Statement

A local grocery store subscribes to an online service that enables their customers to order groceries online. After a customer completes an order, the online service sends a CSV file to the grocery store. The store needs you to write a program that reads the CSV file and prints (to the terminal window) a receipt that lists the purchased items and shows the subtotal, the sales tax amount, and the total.

Assignment

During the prove assignment for lesson 3, you wrote the part of this program that reads and processes two CSV files, one named products.csv that contains a catalog of products and one named request.csv that contains a customer's order. During this assignment (lesson 4), you will add code to finish printing a receipt. Specifically, your program must do the following:

  1. Print the store name at the top of the receipt.
  2. Print the list of ordered items.
  3. Sum and print the number of ordered items.
  4. Sum and print the subtotal due.
  5. Compute and print the sales tax amount. Use 6% as the sales tax rate.
  6. Compute and print the total amount due.
  7. Print a thank you message.
  8. Get the current date and time from your computer's operating system and print the current date and time.

Helpful Documentation

The prepare content for this lesson includes lots of information about calling functions and methods.

This video shows a BYU-Idaho faculty member writing code that prints the current date and time formatted so that users can easily read it.

The datetime module contains the now method, which will get the current date and time from your computer's operating system. To get the current date and time from your computer's operating system, import the datetime module and add this code to your program:

import datetime

# Get the current date and time from the datetime
# module which gets it from the operating system.
current_date_and_time = datetime.datetime.now()

After the computer executes the above code, the variable current_date_and_time will hold the current date and time. You can then add code to call the strftime method, which will format a date and time into a string that is easily readable by a user. For example:

import datetime

# Get the current date and time from the datetime
# module which gets it from the operating system.
current_date_and_time = datetime.datetime.now()

# Format the current date and time to include only
# the day of the week, the hour, and the minute.
string = current_date_and_time.strftime("%A %I:%M %p")

# Print the formatted string.
print(string)

In the previous example, the string sequences that begin with the percent symbol (%) are called format codes. The strftime format codes and their meaning are listed in this document. When executed, the previous example will print the current date and time to the terminal window like this:

> python example.py
Tuesday 01:23 PM

Testing Procedure

Verify that your program works correctly by following each step in this testing procedure:

  1. Run your program and verify that it computes the number of items, subtotal, sales tax, and total as shown in the "Sample Run" section above.
  2. Verify that your program prints a receipt formatted exactly like the one shown in the "Sample Run" section above, except the last line with the date and time will have a different value and may be formatted differently.
