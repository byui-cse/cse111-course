CSE 111 | Programming with Functions

04 Checkpoint: Calling Built-in Functions

Overview

Demonstrate your understanding of calling Python built-in functions and functions from a standard module by completing the following individual checkpoint assignment.

Helpful Documentation

To learn more, you can read about the Python Built-in Functions and the statistics standard module in the official Python documentation.

Assignment

Do the following:

  1. In your text editor, open a new file for a Python program.
  2. At the top of your program, import the statistics module which is a Python standard module.
  3. Copy and paste this Python code that creates a list into your program. 
    heights = [
    1.72, 1.50, 1.96, 2.01, 1.75,
    2.11, 1.60, 1.65, 2.05, 1.50,
    1.80, 1.83, 2.05, 1.79, 1.84
]
  4. Write Python code to call each of these built-in functions and pass the list of heights to each function. Store the value returned from each function in a different variable.
    1. len
    2. min
    3. max
    4. sum
  5. Write Python code to call each of these functions from the statistics module and pass the list of heights to each function. Store the value returned from each function in a different variable.
    1. mean
    2. median
    3. mode
  6. Write code to call the built-int round function to round the value returned from the sum function to and the mean function two digits after the decimal place and store the result in a variable.
  7. Write code to print the value returned from each of the functions.

Sample Run

> python check_solution.py
len: 15
min: 1.5
max: 2.11
sum: 27.16
mean: 1.81
median: 1.8
mode: 1.5

Testing Procedure

Verify that your program works correctly by following each step in this testing procedure:

  1. Run your program. Ensure that your program's output matches the sample run output.

Sample Solution

When your program is finished, view the sample solution for this assignment to compare your solution to that one. Before looking at the sample solution, you should work to complete this checkpoint program first. However, if you have worked on it for at least an hour and are still having problems, feel free to use the sample solution to help you finish your program.
