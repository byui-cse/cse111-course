CSE 111 | Programming with Functions

04 Checkpoint: Calling Functions

Overview

Demonstrate your understanding of calling Python built-in functions and functions from a standard module by completing the following individual checkpoint assignment.

Helpful Documentation

  • Built-in Functions
  • statistics standard module

Assignment

Do the following:

  1. In VS Code, open a new file for a Python program and save it as heights.py.
  2. Copy and paste the following Python code into your program. The code imports the statistics module, which is a Python standard module, and creates a list named heights. 
    import statistics

# A list of people's height in meters.
heights = [
    1.72, 1.50, 1.96, 2.01, 1.75,
    2.11, 1.60, 1.65, 2.05, 1.50,
    1.80, 1.83, 2.05, 1.79, 1.84
]
  3. Write Python code to call each of these built-in functions and pass the list of heights to each function. Store the value returned from each function in a different variable.
    1. len
    2. min
    3. max
    4. sum
  4. Write Python code to call each of these functions from the statistics module and pass the list of heights to each function. Store the value returned from each function in a different variable.
    1. mean
    2. median
    3. mode
  5. Write code to call the built-in round function to round the value returned from the sum function and the mean function to two digits after the decimal place and store the result in a variable.
  6. Write code to print the value returned from each of the functions.

Sample Run

> python check_solution.py
len: 15
min: 1.5
max: 2.11
sum: 27.16
mean: 1.81
median: 1.8
mode: 1.5

Testing Procedure

Verify that your program works correctly by following each step in this testing procedure:

  1. Run your program. Ensure that your program's output matches the sample run output.
