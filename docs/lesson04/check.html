04 Checkpoint: Installing Third-Party Modules

Overview

Use pip to download and install the requests module. Then verify that requests was installed correctly and that it works by running a simple program that uses the requests module to retrieve commodity prices for wool and oranges.

Helpful Documentation

The Python pip command is a computer command that you will use to install the requests module on your computer.

The Python requests module contains functions to send http requests for data to a computer server on the internet and to read the data that the server returns.

Assignment

Do the following:

  1. On the menu bar for VS Code, click "Terminal" and then "New Terminal". This will open a terminal frame at the bottom of the VS Code window. A terminal is a window or frame where a user can type and execute computer commands.
  2. Copy and paste the following command into the terminal frame and execute the command by pressing the Enter key. 
    python -m pip install --user --upgrade pip setuptools wheel
    That command will upgrade pip and several other parts of the Python installation modules so that pip will work correctly.
  3. Install the requests module by copying, pasting, and executing the following command in the terminal frame. 
    python -m pip install --user requests
  4. Download and save this python program and then open it in VS Code.
  5. Run the program that you downloaded in the previous step by clicking the run icon (small green triangle) in the upper right corner of VS Code. If your computer is connected to the internet, the program should run successfully and print prices for Australian wool similar to this:

  6. Examine the commodities.py program in VS Code and note the following which will help you in the team activity and prove assignment for this week:
    1. At line 1, the program imports parts of the standard Python datetime module.
    2. At line 2, the program imports the third-party requests module.
    3. At lines 4 through 24, the program prepares an http request.
    4. At line 27, the program uses the requests module to send the http request to a computer server at quandl.com. Notice in the call to requests.get that the program is using a named parameter, which coincidentally is named params. Named parameters are optional. In this program, the named parameter params causes the quandl.com server to filter the data that it will send to the program.
    5. At line 30, the program checks the status_code of the request. A status code of 200 means the request succeeded.
    6. If the request succeeded, at line 33, the program converts the data that came from the server from JSON data into a dictionary.
    7. At lines 34 through 39, the program prints the data that was returned by quandl.com from the request.
    8. Try to appreciate how simple this Python program is and how easy it is to retrieve data from a remote computer server using the reqests module. In other programming languages and even with other Python modules, retrieving data from a server is substantially more complex and difficult.
  7. At line 6 of the program, change the qcode variable from "ODA/PWOOLC_USD" to "ODA/PORANG_USD". Then run the program again by clicking the run icon (small green triangle). The program should print prices for oranges in the U.S.
