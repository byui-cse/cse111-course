CSE 111 | Programming with Functions

04 Prepare: Calling Third-Party Functions

Because most useful computer programs are very large, programmers divide their programs into parts. Dividing a program into parts makes it easier to write, debug, and understand the program. A programmer can divide a Python program into modules, classes, and functions. In this lesson, you will learn how to call existing functions in third-party modules, and in the next unit, you will learn how to write your own functions.

Here are the Python programming concepts and topics that you should learn this week:

  1. A function is a group of statements (computer commands) that together perform one task.
  2. A Python module is collection of related functions.
  3. Python includes many standard modules, such as csv, datetime, math, random, and textwrap. You can read the documentation for the standard modules in the official online Python documentation
  4. There are many modules that are not part of the standard Python modules. These additional modules are sometimes called third-party modules and are written and maintained by other people as a service to the Python community. Third-party modules are usually available at no cost.
  5. The Python package index (pypi) is an online repository of software that contains more than 200,000 software packages, each of which contains one or more third-party Python modules.
  6. pip is a computer command that you can use to download and install third-party modules. During this course, you will use pip to download and install several third-party modules.
  7. A programmer (you) can save lots of time by using existing modules.

To learn more about installing modules using the pip command, watch this video and read the official pip documentation.
