Problem Statement

Health professionals who are helping a client achieve a healthy body weight, sometimes use two computed values called body mass index and basal metabolic rate.

From the U.S. Centers for Disease Control and Prevention we read, "Body mass index (BMI) is a person's weight in kilograms divided by the square of their height in meters. BMI can be used to screen for weight categories such as underweight, normal, overweight, and obese, that may lead to health problems. However, BMI is not diagnostic of the body fatness or health of an individual." The formula for computing BMI is

bmi =
10000 weight
height2

where weight is in kilograms and height is in centimeters.

Basal metabolic rate (BMR) is the minimum number of calories required to keep your body functioning at rest. BMR is different for women and men and changes with age. The revised Harris-Benedict formulas for computing BMR are

(women) bmr = 447.593 + 9.247 weight + 3.098 height - 4.330 age
(men) bmr = 88.362 + 13.397 weight + 4.799 height - 5.677 age

where weight is in kilograms and height is in centimeters.

Instructions

Download this file: fitness.csv and save it in the same folder where you will save your Python program. Open the file in VS Code and notice that it contains four columns: gender, birthdate, weight (in U.S. pounds), and height (in U.S. inches).

Write a Python program that reads the contents of the fitness.csv file, converts weight from pounds to kilograms, converts inches to centimeters, and calculates age, BMI, and BMR.

Do the following:

  1. Start your program by copying and pasting this code into your program at the top of your Python file. 
    import csv
from datetime import date, datetime
import math

def compute_age(birth):
    """Compute and return a person's age in years.

    param birth: a person's birthdate stored as
        a string in this format: YYYY-MM-DD
    """
    birthdate = datetime.strptime(birth, "%Y-%m-%d").date()
    now = date.today()
    diff = now - birthdate
    years = math.floor(diff.days / 365.25)
    return years
  2. Write a function named kg_from_lb that takes one parameter named lb and converts the value in the parameter to kilograms and returns the kilogram
  3. Write a function named cm_from_in that takes one parameter named inch and converts the value in the parameter from inches to centimeters and returns the centimeter value.
  4. Write a function named body_mass_index that takes two parameters named weight and height. This function must compute and return a person's body mass index. Use the formula for BMI given in the Problem Statement section of this document.
  5. Write a function named basal_metabolic_rate that takes four parameters named gender, weight, height, and age. This function must compute and return a person's basal metabolic rate. Use the formulas for BMR given in the Problem Statement section of this document.
  6. Write a function named main that takes no parameters and does the following:
    1. Prints this header: "Gender, Age (years), Weight (kg), Height (cm), BMI, BMR"
    2. Opens the fitness.csv file for reading
    3. Uses the csv module to get a reader object
    4. Processes each row in the csv file by calling the compute_age, kg_from_lb, cm_from_in, body_mass_index, and basal_metabolic_rate functions.
