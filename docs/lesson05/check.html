05 Checkpoint: Fuel Efficiency

Overview

Demonstrate your understanding of writing your own functions with parameters and then calling those functions by completing the following individual checkpoint assignment.

Problem Statement

Many vehicle owners record the fuel efficiency of their vehicles as a way to track the health of the vehicle. If the fuel efficiency of a vehicle suddenly drops, there is probably something wrong with the engine or drive train of the vehicle. In the United States, fuel efficiency for gasoline powered vehicles is computed as miles per gallon. In most other countries, fuel efficiency is computed as liters per 100 kilometers.

Assignment

Write a program that reads odometer values and fuel amounts from a CSV file and computes and outputs fuel efficiency in both miles per gallon and liters per 100 kilometers.

Do the following:

  1. Download this file: fuel_usage.csv and save it in the same folder where you will save your Python program.
  2. At the top of your program import the csv module.
  3. Write a function named miles_per_gallon that has this header: 
    def miles_per_gallon(start, end, gallons):
    1. The start parameter will contain an odometer value.
    2. The end parameter will contain another odometer value that is larger than the start value.
    3. The gallons parameter will contain a volume amount in U.S. gallons.
    4. This small function must compute and return the fuel efficiency in miles per gallon. The formula for miles per gallon is
      mpg =
      end - start
      gallons
  4. Write another function named lp100k_from_mpg that has this header: 
    def lp100k_from_mpg(mpg):
    1. The mpg parameter will contain a fuel efficiency value in miles per gallon.
    2. This small function must convert the miles per gallon to liters per 100 kilometers and return the converted value. The formula for converting miles per gallon to liters per 100 kilometers is
      lp100k =
      235.215
      mpg
  5. Write another function named main with this header: 
    def main():
    This function must do the following:
    1. Print this header: "Date, Miles/Gallon, Liters/100 Km"
    2. Open the fuel_usage.csv file for reading.
    3. Use the csv module to get a reader object.
    4. Process each row in the csv file by calling the miles_per_gallon and lp100k_from_mpg functions.
    5. Print the date, miles per gallon, and liters per 100 kilometers as comma separated values.
  6. Write a call to the main function.
