08 Checkpoint: Run a Program that Handles Exceptions

Overview

Complete this individual checkpoint assignment to improve your understanding of how exceptions are handled in a Python program.

Assignment

Do the following:

  1. Download and save the get_line.py program and then open it in VS Code.
  2. get_line.py is a simple program that asks a user to input the name of a text file and a line number. Then get_line.py prints the text that is in the text file on the user requested line number.
  3. Notice in get_line.py, the try block at the beginning of the main fuction. Notice also the four except blocks, each handling a different type of exception.
  4. Run the get_line.py program five times and enter the input shown in the Sample Run section below. For each of the first four times that you run the program, find the lines of code in get_line.py that handled the exception that was raised.
