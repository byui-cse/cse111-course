08 Checkpoint: Dictionaries

Overview

Complete this individual checkpoint assignment to improve your understanding of how to use a dictionary in a Python program.

Helpful Documentation

  • The prepare content for this lesson explains how to find items in a dictionary.
  • The prepare content for lesson 7 explains how to retrieve an element from a list.
