def cels_from_fahr(fahr):
    """Convert a Fahrenheit temperature to
    Celsius and return the Celsius temperature.
    """
    cels = (fahr - 32) * 5 / 9
    return round(cels, 1)

    At lines 10 - 12 in the example code above, there is a for each loop that converts each Fahrenheit temperature to Celsius and then appends the Celsius temperature into a new list. Writing a for each loop like this is the traditional way to process all the elements in a list. However, Python also includes a function named map that will process all of the elements in a list and return a new list that contains the results. The map function contains a loop inside of it, so that when a programmer uses the map function, he doesn't need to write a loop.     Notice that the second example (above), doesn't contain a for each loop. Instead, at line 9, it contains a call to the map function. Remember that the map function has a loop inside of it, so that the programmer who calls map, doesn't have to write the loop. Notice also at line 9, that the first argument to the map function is the name of the cels_from_fahr function. In other words, at line 9, we are passing the cels_from_fahr function into the map function, so that map will call cels_from_fahr for each element in the fahr_temps list.

    The map function is just one example of passing a function as an argument into a function. When writing a Python program there are many other opportunities to pass a function as an argument into another function.
