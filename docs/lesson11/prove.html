CSE 111 | Programming with Functions

11 Prove Milestone: Significant Python Program

Overview

Prove that you can develop a significant Python program with multiple functions that you verify are correct with test functions and pytest.

Assignment

During this lesson and the next two lessons, write a significant Python program that does something that interests you. You may choose one of the programs from the list below, or you may write a program that you design. However, if you intend to write a program that is not in the list below, you must first propose your program to your instructor.

  • A program that uses pandas to process a dataset like the water meter reading dataset and produces charts that help people to visualize information about the dataset.
  • A program with a complete graphical user interface (GUI), including windows, frames, labels, text fields, and buttons. Use tkinter or kivy.
  • A program that stores and retrieves data from a relational database.
  • A program that stores and retrieves data from a Firebase database.
  • A program that graphically simulates the spread of a disease like the simulations in this Washington Post article. Your program should include variables that can be changed, such as movement rate, probability of transmission, length of contagiousness, length of illness, and probability of death.
  • A program that reads an English text file and respells the words to more closely match their pronunciation. For more information, see the English-language spelling reform article at Wikipedia.

    • Organization

    As you develop your program, be certain to divide the program into functions. Each function should perform a single task and should be appropriately named.

    Testing Procedure

    As you develop your program, write test functions that test your program functions. Run the test functions using pytest to verify the correctness of your program functions.

    Submission

    At the end of each lesson: 11, 12, and 13, you must submit the following:

    • the number of hours that you worked on your program
    • a description of the progress that you made on your program
    • the source code for your program and test functions
