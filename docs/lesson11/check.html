11 Checkpoint: Functional Programming

Overview

Complete this individual checkpoint assignment to reinforce in your mind the concept that a function can be passed as an argument to another function.

Problem Statement

Phone numbers in the U.S. are often stored as ten digits and two dashes in this format: "ddd-ddd-dddd" where each d is a digit. The first three digits are called the area code. For example, for many years, 208 was the area code for all phone numbers from the state of Idaho and 801 and 435 were the two area codes for phone numbers in the state of Utah. Some people when entering their phone number will not enter the area code and assume that other people will know the correct area code.

Helpful Documentation

The prepare content for this lesson explains the pandas apply function.

Assignment

Write a program that reads data from a CSV file into a pandas DataFrame. Your program must add the area code "208-" to all phone numbers in the DataFrame that don't have an area code. Do the following:

  1. Download these two files: pupils.csv and add_area_code.py and save them in the same folder.
  2. Open the pupils.csv file in VS Code and examine the phone numbers in the "phone" column. Notice that some of the phone numbers do not contain an area code and contain only seven digits.
  3. Open the add_area_code.py file in VS Code. Read the comments in the program and then add code to the program so that it does the following:
    1. Read the data in pupils.csv into a DataFrame.
    2. Add the area code "208-" to the phone numbers that do not contain an area code.
    3. Print the DataFrame with the corrected phone numbers.

Testing Procedure

Verify that your program works correctly by following each step in this testing procedure:

  1. Download, examine, and run this Python file named test_add_area_code.py that contains pytest test functions. Verify that all the test functions pass. If they don't pass, fix the mistakes in your code until they do pass.
  2. Run your program which should print the DataFrame. Examine the phone numbers in the DataFrame and verify that all phones have an area code.

Ponder

In the main function, look at the line of code that calls the pandas apply function. Think about the fact that the apply function will call the add_area_code function once for each phone number in the "phone" column, and the add_area_code function will add Idaho's area code "208-" at the beginning of the phone number if necessary.
