11 Team Activity: Functional Programming

Instructions

Arrange a one hour synchronous meeting with your team for this activity. Online students should coordinate a video-sharing meeting. Campus students will use class time for this meeting. You should prepare for this meeting by completing the preparation material and the individual checkpoint assignment beforehand.

Problem Statement

In the United States, minor students are required by law to attend school. There are 13 grade levels for students. Usually students are assigned to a grade level based on their age on a specific cutoff date, which is often October 1. Ages and grade levels are shown in the following table:

Age
(years)		 Grade Level
5 kindergarten
6 first
7 second
8 third
9 fourth
10 fifth
11 sixth
12 seventh
13 eighth
14 freshman
15 sophomore
16 junior
17 senior

As an example, a student who is 7 years old on October 1 would be assigned to second grade.

Helpful Documentation

  • The prepare content for this lesson explains Python lambda functions and the pandas apply function.
  • The official pandas reference for the apply function briefly explains the args named parameter.
  • The prepare content for the first pandas lesson contains sample code for adding a new column to a DataFrame and for counting values in a DataFrame.

Assignment

Do the following:

  1. Download these two files: students.csv and grade_level.py and save them in the same folder.
  2. Open the students.csv file in VS Code and notice that it has six columns: givenName, surname, birthdate, parentGivenName, parentSurname, phone. Your program will add two more columns named ageAtCutoff and gradeLevel.
  3. Open the grade_level.py file in VS Code. Read the comments in the program and then complete the program by replacing each pass statement with the correct Python code.

Core Requirements

  1. Your finished grade_level.py program must contain at least the following:
    1. four regular functions named main, add_columns, year_diff, and grade_level_counts
    2. one lambda function
    3. two calls to the pandas apply function
  2. Your program must add to the original data frame two columns named ageAtCutoff and gradeLevel and must print the data frame.
  3. Your program must create and print a new pandas Series that contains a count of the number of students assigned to each grade level.

Stretch Challenges

If your team finishes the core requirements in less than an hour, complete one or more of these stretch challenges. Note that the stretch challenges are optional.

  1. Before printing the students data frame, sort it by ageAtCutoff, surname, and givenName.
  2. Before printing the grade level counts series, sort it by the normal ordering of grade levels which is shown in this assignment in the table above.

Testing Procedure

Verify that your program works correctly by following each step in this testing procedure:

  1. Download, examine, and run this Python file named test_grade_level.py that contains pytest test functions. Verify that all the test functions pass. If they don't pass, fix the mistakes in your code until they do pass.
  2. Run your program. Ensure that your program's output is similar to the sample run output.
