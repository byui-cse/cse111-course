02 Checkpoint: Review Selection

Overview

Demonstrate your understanding of selection (if statements) and logic by completing the following individual checkpoint assignment.

Problem Statement

You work for a retail store that wants to increase sales on Tuesday and Wednesday, which are the store's slowest sales days. On Tuesday and Wednesday, if a customer's subtotal is greater than $50, the store will discount the customer's subtotal by 10%.

Assignment

Write a Python program named discount.py that takes a customer's subtotal as input. If the subtotal is greater than $50 and today is Tuesday or Wednesday, your program must subtract 10% from the subtotal. Your program should then compute and output the total amount due by adding sales tax of 6% to the subtotal.

Your program must not ask the user to enter the day of the week. Instead, it must get the day of the week from your computer's clock. At the top of your program, import the datetime module like this:

import datetime

Then within your program, use this Python code that will get the current day of the week from your computer's clock:

weekday = datetime.datetime.now().isoweekday()

If you use the above code, the variable weekday will hold 1 if today is Monday, 2 if today is Tuesday, and so on to 7 if today is Sunday.

Sample Runs

Run on a Monday

> python check_solution.py
Please enter the subtotal: 42.75
Total: 45.31

> python check_solution.py
Please enter the subtotal: 55.20
Total: 58.51

Run on a Tuesday

> python check_solution.py
Please enter the subtotal: 42.75
Total: 45.31

> python check_solution.py
Please enter the subtotal: 55.20
Total: 52.66

Testing Procedure

Verify that your program works correctly by following each step in this testing procedure:

  1. If today is any day other than Tuesday or Wednesday, run your program using the input shown in the "Run on a Monday" section above. Ensure that your program's output matches the sample run output. If today is Tuesday or Wednesday, run your program using the input in the "Run on a Tuesday" section.
  2. Is there a simple way to test your program for all the days of the week without testing on seven consecutive days and without changing your computer's system date? Hint: In your program immediately after this line of code: 
    weekday = datetime.datetime.now().isoweekday()
    temporarily add a line of code like this one: 
    weekday = 3

However, if you have worked on it for at least an hour and are still having problems, feel free to use the sample solution to help you finish your program.
