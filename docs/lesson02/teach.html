Problem Statement

While entering input for a program, users ocassionally make mistakes and enter invalid or incorrect input. A good program recovers from user mistakes, informs users about mistakes in clear polite language, and gives users an opportunity to correct their mistakes.

Helpful Documentation

The built-in Python len function returns the number of characters in a string.

The string isdecimal method checks the characters in a string to see if they are all decimal digits or not.

# Create a dictionary that holds product numbers and names.
products = {
    38500: "Backpacking Tent",
    27840: "Sit-Stand Desk",
    37992: "Mouse",
    24458: "Subwoofer",
    38323: "USB Cable",
    14392: "Monitor",
    11103: "Sleeping Bag",
    41023: "Cat-6 Network Cable",
    40246: "HDMI Cable",
    23047: "Keyboard",
    20319: "Keyboard",
    32061: "Mouse"
}

# Get a product number from the user.
prod_num = input("Please enter a product number: ")

# Convert the product number from a string to
# an integer, so that this program can find the
# product number in the products dictionary.
prod_num = int(prod_num)

# Find the corresponding product name and print it.
prod_name = products[prod_num]
print(f"{prod_num}: {prod_name}")

Core Requirements

Do the following:

  1. Add a loop around the call to the input function on line 19, so that if a user enters an invalid product number, your program will print an error message and prompt the user again.
  2. Within the loop and after the call to the input function, add code that validates the user input.
    1. If the user enters any character that is not a decimal digit, your program must print "Invalid character in the product number" (without the quotes).
    2. If the user enters fewer than five decimal digits, your program must print "Invalid product number: too few digits" (without the quotes).
    3. If the user enters more than five decimal digits, your program must print "Invalid product number: too many digits" (without the quotes).
  3. If the user enters a valid product number (five decimal digits), your program must continue and find the product number in the products dictionary and print the corresponding product name.

Stretch Challenges

If your team finishes the core requirements in less than an hour, complete one or more of these stretch challenges. Note that the stretch challenges are optional.

  1. After the user input validation code and immediately before the line of code that finds a product number in the products dictionary, add code to your program that uses an if statement with the Python in operator to check if a product number is in the products dictionary.
    1. If the product number is not in the products dictionary, your program must print "No such product" (without the quotes).
    2. Otherwise, your program must print the product number and name.
