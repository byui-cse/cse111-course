Problem Statement

Many schools, businesses, and other institutions ask their stakeholders to complete surveys. Some of the questions on the surveys ask the stakeholders to rate some characteristic on a scale. For example, BYU-Idaho asks its students this question, "What is your overall rating of this course?" Students respond by selecting one of these choices:

  1. Very Poor
  2. Poor
  3. Fair
  4. Good
  5. Very Good
  6. Excellent
  7. Exceptional

The results of this question are summarized by assigning a number from 1 (very poor) to 7 (exceptional) to each response and then computing the average of those numbers. Courses that receive an average overall rating less than 4 are improved by faculty members and curriculum designers.

Assignment

The responses.txt file contains the numeric rating responses from one question on a survey. Download the responses.txt file and save it in the same folder where you will save your Python program. Open the file in VS Code or another text editor and examine it. Notice that each line in the file contains one integer. Each integer represents one person's response to one question.

Core Requirements

Write a program that does the following:
  1. Opens the responses.txt file for reading.
  2. Reads each response from the file and converts each response from a string to a number and stores all responses in a list of numbers.
  3. Computes the average value of all the responses and prints the number of responses and the average value of the responses.

Sample Run

> python teach_solution.py
responses.txt
There are 25 responses with an average value of 4.0
14 of the responses are greater than the average.

Stretch Challenges

If your team finishes the core requirements in less than an hour, complete one or more of these stretch challenges. Note that the stretch challenges are optional.

What happens if a stakeholder doesn't answer a question on a survye? What rating should be used for a non answer? Perhaps non answers should be skipped and not computed in the average. In the responses.txt file, several of the lines contain the number -1. The -1 indicates that the stakeholder didn't answer the question. Modify your program so that it does the following:

  1. Ignores or skips all responses that are less than zero (0).
  2. Counts the number of responses that are greater than the average response and prints that count.

Sample Run

> python teach_solution.py
responses.txt
There are 21 responses with an average value of 4.95
14 of the responses are greater than the average.
