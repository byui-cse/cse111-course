02 Prove: Canadian Provinces and Territories

Overview

Prove that you can write a Python program that reads a text file, replaces abbreviations with full proper names, writes a text file.

Problem Statement

The nation of Canada has ten provinces and three territories. The locations.txt file contains survey responses for a question that asked Canadians where they lived. Unfortunately, the creator of the survey allowed people to respond with the name or the postal abbreviation of their home province or territory. Allowing both names and abbreviations makes summarizing the data difficult.

Assignment

Download the locations.txt file and save it in the same folder where you will save your Python program. Open the file in VS Code or another text editor and examine it. Notice that each line in the file contains either a name or abbreviation of a province or territory in Canada. Write a Python program that does the following:

  1. Opens the locations.txt file for reading.
  2. Opens a text file named loc_names.txt for writing.
  3. Reads each line from locations.txt. For each line:
    1. If the line contains an abbreviation, your program must convert the abbreviation to the corresponding name.
    2. Write each name to the loc_names.txt file.

Helpful Documentation

To convert abbreviations to names, your program can use a dictionary or a group of if, elif, else statements inside the loop that reads each line from locations.txt.

Testing Procedure

Verify that your program works correctly by following each step in this testing procedure:

  1. Run your program.
  2. Your program should create the loc_names.txt file. Open the file in VS Code.
  3. Verify that loc_names.txt contains the same data in the same order as locations.txt except that each abbreviation has been replaced with the corresponding proper name.
