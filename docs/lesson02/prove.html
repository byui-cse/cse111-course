02 Prove: Dictionaries

Overview

Prove that you can write a simple Python program that creates a dictionary from a CSV file and allows a user to look up values in the dictionary.

Problem Statement

products.csv is a CSV file that contains information about products sold by an online store. Write a program that reads products.csv into a Python dictionary and allows a user to look up product information by the product number.

Assignment

Each row in the products.csv products.csv file contains five columns: product #, name, description, category, and price. (Note that after you download products.csv you can view its contents by opening it in VS Code.) Your program must do the following:

  1. Read the product information from products.csv into a Python dictionary.
  2. Prompt the user for a product number.
  3. Write the prompt code inside of a loop so that if a user enters an invalid product number, your program will print an error message and prompt the user again.
  4. All product numbers in products.csv are composed of five decimal digits. Your program must verify that the user entered a valid product number.
    1. If the user enters any character that is not a decimal digit, your program must print "Invalid character in product number" (without the quotes).
    2. If the user enters fewer than five decimal digits, your program must print "Invalid product number: too few digits" (without the quotes).
    3. If the user enters more than five decimal digits, your program must print "Invalid product number: too many digits" (without the quotes).
  5. Your program must look up the product number in the dictionary and print the corresponding product name and price separated by a comma and a space (", ").
  6. Your program should not print the product description and category.
  7. If the user entered a valid product number (five digits) that does not exist in products.csv, your program must print "No such product" (without the quotes).
