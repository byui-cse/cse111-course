02 Prepare: Review Python

Here are the Python programming concepts and topics from CSE 110 that you should review this week.

  1. In Python, we use if statements to cause the computer to make decisions. if statements are also called selection because the computer selects one group of statements to execute and skips the other group of statements.
  2. Each if statement may have an else statement. We can combine else and if into the keyword elif.
  3. There are six comparison operators that we can use in an if statement: less than (<), less than or equal (<=), greater than (>=), greater than or equal (>=), equal to (==), and not equal to (!=). Python also includes two membership operators: in and not in.
  4. Python includes two logic operators: and, or that we can use to combine two comparisons. Python also includes the logical not operator.
  5. We can cause a computer to repeat a group of statements by writing for and while loops. A for loop iterates over a range such as range(3, 10) or a sequence such as a list. A while is more flexible than a for loop and repeats while some condition is true. A break statement causes a loop to end early.
  6. We can store several items in a Python tuple. Tuples are immutable, meaning that after a tuple is created, it cannot be changed.
  7. We can store many items in a Python list. Lists are mutable, meaning they can be changed. We can add an item to a list using insert and append. We can replace an item in a list and retrieve an item from a list using the index operators ([ and ]). We can remove an item from a list using del, pop, and remove. We can find an item in a list using index, and we can determine the number of items in a list using the len function.
  8. We can store many items in a Python set. Sets are mutable, meaning they can be changed. A set is different from a list. A list allows duplicate items to be stored in it, but a set doesn't allow duplicate items. Items in a set are unique.
  9. We can also store many items in a Python dictionary. Dictionaries are mutable, meaning they can be changed. In a dictionary, items are stored as key value pairs and we can easily write code to find a value by its key.
  10. Python has a module named csv that includes functionality to read and write files with comma separated values (CSV files).
