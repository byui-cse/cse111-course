02 Prepare: Review Python

Concepts

Here are the Python programming concepts and topics from CSE 110 that you should review during this lesson.

  1. In Python, we use if statements to cause the computer to make decisions; if statements are also called selection because the computer selects one group of statements to execute and skips the other group of statements.
  2. There are six comparison operators that we can use in an if statement:
    < less than
    <= less than or equal
    > greater than
    >= greater than or equal
    == equal to
    != not equal to
    Here is example code that tests if a number is greater than 500. 
    # Get an account balance as a number from the user.
balance = float(input("Please enter the account balance: "))

# If the balance is greater than 500, then
# compute and add interest to the balance.
if balance > 500:
    interest = balance * 0.03
    balance += interest
    If you have written programs in other programming languages such as JavaScript, Java, or C++, you always used curly braces to mark the start and end of the body of an if statement. However, notice in the example code above that if statements in Python do not use curly braces. Instead, we type a colon (:) after the comparison of the if statement and then indent all the statements that are in the body of the if statement. The body of the if statement ends with the first line of code that is not indented. It may seem strange to not use curly braces to mark the start and end of the body of an if statement. However, the Python way forces us to write code where the indentation matches the functionality or in other words where the way we write the code matches the way that the computer will execute the code.
  3. Python also includes a membership operator: in. We can use the in operator to determine if a tuple, list, set, or dictionary contains a value. For example: 
    names = {"Smith", "Lopez", "Marsh", "Olsen", "Vasiuk"}

family = input("Please enter your family name: ")
if family in names:
    print("Your family name is in the set.")
else:
    print("Your family name is NOT in the set.")
  4. Each if statement may have an else statement. We can combine else and if into the keyword elif. For example: 
    # Get the cost of an item from the user.
cost = float(input("Please enter the cost: "))

# Determine a discount rate based on the cost.
if cost < 100:
    rate = 0.10
elif cost < 250:
    rate = 0.15
elif cost < 400:
    rate = 0.18
else:
    rate = 0.20

# Compute the discount amount and the discounted cost.
discount = cost * rate
cost -= discount

# Print the discounted cost for the user to see.
print(f"After the discount, you will pay {cost}")
  5. Python includes two logic operators: and, or that we can use to combine two comparisons. Python also includes the logical not operator. Notice in Python that the logical operators are literally the words: and, or, not and not symbols as in other programming languages: 
    if driver >= 54 or (driver >= 32 and passenger >= 54):
    message = "Enjoy the ride!"
  6. We can cause a computer to repeat a group of statements by writing for and while loops. A for loop iterates over a range such as range(3, 10) or a sequence such as a list. A while loop is more flexible than a for loop and repeats while some condition is true. The next code example shows four loops. Notice that just like if statements, in Python the body of a loop starts and ends with indentation. 
    # Count from zero to ten by ones.
for i in range(10):
    print(i)

# Count from zero to ten by twos.
for i in range(0, 10, 2):
    print(i)

# Count from 100 down to 70 by three.
for i in range(100, 70, -3):
    print(i)

# Print each value in a set.
names = {"Smith", "Lopez", "Marsh", "Olsen", "Vasuik"}
for family in names:
    print(family)
  7. A break statement causes a loop to end early. In the next code example, there is a for loop that asks the user to input ten numbers one at a time. However, the loop will terminate early if the user enters a zero (0). 
    # Get ten or fewer numbers from the user and add them
# together. Notice that this loop uses underscore (_) as
# the counting variable, which is a valid variable name.
# Many programmers use underscore to indicate that the
# variable will not be used within the body of the loop.
sum = 0
for _ in range(10):
    n = float(input("Please enter a number: "))
    if n == 0:
        break
    sum += n
print(sum)
  8. The built-in open() function opens a file for reading or writing. The file.read() method reads characters or bytes from a file. The file.readline() method reads one line of text from a text file. The file.readlines() method reads all the lines of text from a text file and returns the lines in a list. 
    # Open a file named contract.txt and store a
# reference to the opened file in a variable
# named infile.
with open("contract.txt", "rt") as infile:

    # Read the first line of text from the file.
    first = infile.readline()

    # Read all remaining lines of
    # text from the file into a list.
    remain = infile.readlines()
  9. Many computer systems import and export data in CSV files. CSV is an acronym for comma separated values. A CSV file is a plain text file that contains tabular data where each row is on a separate line and each cell (column) is separated by a comma. The following example is the contents of a CSV file that stores information about some dental offices in Rexburg, Idaho. Notice that the first row of the file contains column headings, the next four rows contain information about four dental offices, and that each row contains three columns separated by commas. 
    Company Name,Address,Phone Number
Eagle Rock Dental Care,556 Trejo Suite C,208-359-2224
Apple Tree Dental,33 Winn Drive Suite 2,208-359-1500
Rockhouse Dentistry,106 E 1st N,208-356-5600
Cornerstone Family Dental,44 S Center Street,208-356-4240
    Python has a module named csv that includes functionality to read and write data from CSV files. The following example code shows how to open a CSV file and use the csv module to read and print each row. 
    import csv

# Open a file named dentists.csv and store a
# reference to the opened file in a variable
# named infile.
with open("dentists.csv", "rt") as infile:

    # Use the csv module to create a reader
    # object that will read from the open file.
    reader = csv.reader(infile)

    # Read each row one at a time and
    # print the contents of each row.
    for row in reader:
        company = row[0]
        address = row[1]
        phone = row[2]
        print(company, address, phone)
    You could use this example code in a program that you write, but of course, you could modify the for loop in the code to do much more than simply print the contents of each row.
