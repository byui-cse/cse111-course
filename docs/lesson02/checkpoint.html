CSE 111 | Programming with Functions

02 Checkpoint:

Overview

Demonstrate your understanding of decisions, if statements, and logic by completing the following individual checkpoint assignment.

Assignment

You work for a retail store that wants to increase sales on Tuesday and Wednesday, which are the store’s slowest sales days. Write a defining table and then a program that takes a customer’s subtotal as input and if the subtotal is greater than $50 and today is Tuesday or Wednesday, subtracts 10% from the subtotal. Your program should then compute and output the total amount due by adding sales tax of 6% to the subtotal. Within your program, use this JavaScript code that will get the current day of the week from your computer’s clock:

    let dayOfWeek = new Date().getDay();

If you use the above code, the variable dayOfWeek will hold 0 if today is Sunday, 1 if today is Monday, and so on to 6 if today is Saturday.
