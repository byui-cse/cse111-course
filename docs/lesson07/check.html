07 Checkpoint: Testing Functions

Overview

Complete this individual checkpoint assignment to demonstrate your ability to verify the correctness of functions by writing test functions and using pytest.

Helpful Documentation

This short video shows a BYU-Idaho faculty member using the pip module to install other Python modules.

This short video shows a BYU-Idaho faculty member writing two test functions and using pytest to run them.

If you wish to learn more about installing modules using the pip module, read the Basic usage section of the official pip reference.

The prepare content for this lesson explains how to use pytest, assert, and approx to automatically verify that functions are correct. It also contains an example test function and links to additional documentation about pytest.

Assignment

Write two test functions that test previously written functions. Use pytest to run the test functions.

Do the following:

  1. Download the words.py Python file and save it in the same folder where you will save your Python test program. Then open the downloaded file in VS Code.
  2. On the menu bar for VS Code, click "Terminal" and then "New Terminal." This will open a terminal frame at the bottom of the VS Code window. A terminal is a window or frame where a user can type and execute computer commands.
  3. Copy and paste the following command into the terminal frame and execute the command by pressing the Enter key. This command will upgrade pip and several other parts of the Python installation modules so that pip will work correctly. 
    python -m pip install --user --upgrade pip setuptools wheel
    • If your computer is running the Mac OS, you must use the python3 command instead of the python command. In other words, after you copy and paste the above command, but before you press the enter key, change python to python3.
  4. Install the pytest module by copying, pasting, and executing the following command in the terminal frame. 
    python -m pip install --user pytest
    • If your computer is running the Mac OS, you must use the python3 command instead of the python command. In other words, after you copy and paste the above command, but before you press the enter key, change python to python3.
  5. Notice the downloaded words.py file contains two small functions named: prefix and suffix. Notice also that each function has a documentation string that describes what the function does. Read the documentation strings for both functions.
  6. In VS Code open a new Python file and save it with the name test_words.py in the same folder where you saved the downloaded words.py file.
  7. Copy and paste these import statements at the top of the test_words.py file: 
    from words import prefix, suffix
import pytest
  8. In test_words.py write a function named test_prefix that takes no parameters and contains assert statements for these nine calls to prefix:
    Parameters Return
    Value
    s1 s2
    "" "" ""
    "" "correct" ""
    "clear" "" ""
    "happy" "funny" ""
    "cat" "catalog" "cat"
    "dogmatic" "dog" "dog"
    "jump" "joyous" "j"
    "unwise" "ungrateful" "un"
    "Disable" "dIstasteful" "dis"
  9. In test_words.py write a function named test_suffix that takes no parameters and contains assert statements for these nine calls to suffix:
    Parameters Return
    Value
    s1 s2
    "" "" ""
    "" "correct" ""
    "clear" "" ""
    "angelic" "awesome" ""
    "found" "profound" "found"
    "ditch" "itch" "itch"
    "happy" "funny" "y"
    "tired" "fatigued" "ed"
    "swimming" "FLYING" "ing"
  10. At the bottom of your test_words.py file, write a call to the pytest.main function like this: 
    # Call the main function that is part of pytest so that
# the test functions in this file will start executing.
pytest.main(["-v", "--tb=no", "test_words.py"])
  11. Save your test_words.py file and run it by clicking the green run icon in VS Code.
