Overview

Write a Python program that calls built-in functions and standard functions.

Project Description

In the spring of 2019, several BYU-Idaho students created a small Android app to help people learn vocabulary words in a foreign language. This app stores the vocabulary words in language pairs in a centralized firebase database. Your assignment during lessons 3 and 4 is to write a Python program that runs on your computer and enables a user to do the following:

  1. select a language pair
  2. list all of the words stored in the database for that language pair
  3. find a word and its translation in the language pair
  4. add a new word and its translation to the language pair

During lesson 3, you will write half of this Python program. This half will display menus and allow the user to choose commands from the menu but will not communicate with the firebase database. During lesson 4, you will add the code to comminucate with the database.

At the top of your program, add these two lines of Python code. Use the list in these two lines to generate the menu for the user to choose a language pair.

langs = ["ENtoRU", "ENtoPT", "PTtoEN", "PTtoRU"]

Within your code, you will call these built-in functions: int, input, len, print, range, and str. Your code will also call these standard string functions (methods) .lower, .strip, and .upper. In addition, your code will include several loops and will use lists.
