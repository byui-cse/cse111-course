07 Prove Milestone: Molar Mass Calculator

Overview

Prove that you can write a Python program and write and run test functions to help you find and fix mistakes.

Problem Statement

The molar mass of a substance is the mass in grams of 1 mole of the substance (grams / mole). In chemistry, a molar mass calculator is a program that computes the molar mass of a substance. To use a molar mass calculator, a chemist enters the formula for a molecule such as H2O or C6H12O6. The calculator splits the formula into elements and for each element in the formula does the following:

  1. Sums the number of atoms of each element in the formula
  2. Looks up the atomic mass of each element
  3. Multiplies the number of atoms by the atomic mass
  4. Adds the product into the molar mass of the molecule

Assignment

During this lesson and the next lesson, you will write and test a molar mass calculator. During this lesson you will complete about half of the task, including writing a function, writing two test functions, running four test functions, and finding and fixing any mistakes that cause test failures.

Do the following:

  1. Download these two Python files: chemistry.py and test_chemistry.py, and save them in the same folder where you will save your Python program. Then open both downloaded files in VS Code.

Testing Procedure

Verify that the Python code in chemistry.py and test_chemistry.py works correctly by using pytest to run the four test functions in test_chemistry.py.

  1. If test_names fails, carefully read the failure message from pytest. Ask yourself, "Given this failure message, which function is incorrect, the test_names function or the get_name function?" Fix the function that has a mistake.
  2. If test_atomic_masses fails, carefully read the failure message from pytest. Ask yourself, "Given this failure message, which function is incorrect, the test_atomic_masses function or the get_atomic_mass function?" Fix the function that has a mistake.
  3. test_parse should not fail because it is supposed to remain unfinished until the next prove assignment. If test_parse fails when you test it, review it and ensure that it contains no code except comments and a pass statement.
  4. If test_molar_mass fails, review and debug the code that you wrote in the molar_mass function in the chemistry.py file.
