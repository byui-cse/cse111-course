07 Prove: Write a Molecular Mass Calculator

Overview

Prove that you can write a function and write and run test functions to help you find and fix mistakes.

Problem Statement

The molar mass of a substance is the mass in grams of 1 mole of the substance. In chemistry a molar mass calculator is a program that computes the molar mass of a substance. To use a molar mass calculator, a chemist enters the formula for a substance such as H2O or C6H12O6. The calculator splits the formula into elements and for each element in the formula does the following:

  1. sums the number of atoms of the element in the formula
  2. looks up the molar mass (also called atomic mass) of the element
  3. multiplies the number of atoms by the molar mass
  4. adds the product into the molar mass of the molecule

Then you will run the four test functions and find and fix any mistakes that cause test failures.

Do the following:

  1. Download and save chemistry.py

  3. Write the molecular_mass function in chemistry.py
  4. Use pytest to run the tests. Fix mistakes, if any, in get_name, get_atomic_mass, parse, and molecular_mass.
