07 Team Activity: Testing and Fixing Code

Overview

Arrange a one-hour synchronous meeting with your team for this activity. Online students should coordinate a video-sharing meeting. Campus students will use class time for this meeting. You should prepare for this meeting by completing the preparation material and the individual checkpoint assignment beforehand.

Problem Statement

Writing and running test functions often helps a software developer find mistakes in code. Use test functions and pytest to find the mistakes in three previously written functions. Then fix those mistakes so that the test functions pass.

Instructions

Download and save this names.py Python file and then open it in VS Code. Notice that the downloaded file contains three small functions named: full_name, given_name, and family_name. Notice also that each function has a documentation string that describes what the function does. Read the documentation strings for both functions.

The functions in the file that you downloaded (names.py) may contain some mistakes. Write a Python test program that tests three functions in names.py. Use pytest to run the test program and use the output from pytest to help you find and fix any mistakes in names.py.

Do the following:

  1. At the top of your test program, add this statement to import the three functions that are in names.py: 
    from names import given_name, family_name, full_name
  2. Write a function named test_full_name that takes no parameters. Write assert statements inside this function to test the value returned from the full_name function. If you are not sure what the full_name function does or how to test it, read the documentation string that is at the top of the full_name function in the names.py file.
  3. Write a function named test_given_name that takes no parameters. Write assert statements inside this function to test the value returned from the given_name function.
  4. Write a function named test_family_name that takes no parameters. Write assert statements inside this function to test the value returned from the family_name function.
  5. Execute this command in the terminal frame of VS Code to run your test program: 
    > python -m pytest <your_python_filename>.py
  6. Read the output from pytest and your test program. Use the output to help you find and fix any errors that are in the full_name, given_name, and family_name functions.
  7. Repeat steps 5 and 6 until you have found and fixed all the mistakes and your tests pass.
