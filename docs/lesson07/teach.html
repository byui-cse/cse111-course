CSE 111 | Programming with Functions

07 Team Activity: Testing and Fixing Code

Instructions

Arrange a one hour synchronous meeting with your team for this activity. Online students should coordinate a video-sharing meeting. Campus students will use class time for this meeting. You should prepare for this meeting by completing the preparation material and the individual checkpoint assignment beforehand.

Problem Statement

Writing and running test functions often helps a software developer find mistakes in code.

Assignment

Write three test functions that test three previously written functions. Use pytest to run the test functions and find and fix the mistakes, if any, that are in three previously written functions. Do the following:

  1. Download this Python file: names.py and save it in the same folder where you will save your Python test program. Then open the downloaded file in VS Code. Notice that the downloaded file contains three small functions named: full_name, given_name, and family_name. Notice also that each function has a documentation string that describes what the function does. Read the documentation strings for all three functions.
  2. The functions in the file that you downloaded (names.py) may contain some mistakes. Write a Python test program named test_names.py that tests the three functions in names.py. Save your test_names.py file in the same folder where you saved the names.py file.
  3. At the top of your test_names.py file, add this statement to import the three functions that are in names.py: 
    from names import given_name, family_name, full_name
  4. In test_names.py write a function named test_full_name that takes no parameters. Write assert statements inside this function to test the value returned from the full_name function. If you are not sure what the full_name function does or how to test it, read the documentation string that is at the top of the full_name function in the names.py file.
  5. In test_names.py write a function named test_given_name that takes no parameters. Write assert statements inside this function to test the value returned from the given_name function.
  6. In test_names.py write a function named test_family_name that takes no parameters. Write assert statements inside this function to test the value returned from the family_name function.
  7. Execute this command in the terminal frame of VS Code to run your test program: 
    > python -m pytest test_names.py
  8. Read the output from pytest and use the output to help you find and fix any errors that are in your test functions or the full_name, given_name, and family_name functions.
  9. Repeat steps 5 and 6 until you have found and fixed all the mistakes and your three test functions pass.

Core Requirements

  1. Write test_full_name so that it tests full_name with various names, including long names, short names, and hyphenated names.
  2. Write test_given_name so that it tests given_name with various names, including long names, short names, and hyphenated names.
  3. Write test_family_name so that it tests family_name with various names, including long names, short names, and hyphenated names.

Stretch Challenges

  1. Fix the mistakes in the full_name function.
  2. Fix the mistakes in the given_name function.
