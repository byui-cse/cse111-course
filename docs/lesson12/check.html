12 Checkpoint: Modifying a List with Object Oriented Programming

Overview

Complete this individual checkpoint assignment to demonstrate your ability to write object oriented code.

Problem Statement

There are several types of commands that are commonly found in object oriented code. Three of those types of commands are:

  1. Creating objects; example: 
    obj = date.today()
  2. Accessing the attributes of an object using dot notation; example: 
    y = obj.year
  3. Calling the methods of an object using dot notation; example: 
    obj.replace(year=2025)

  • More about lists

Assignment

Write a small Python program that demonstrates object oriented programming by modifying a list. Do the following:

  1. Open a new blank file in VS Code and save it as list_object.py
  2. Copy and paste this code at the top of your list_object.py program: 
    def main():
    fruit = ["pear", "banana", "apple", "mango"]
    print(f"original: {fruit}")
  3. Add code to reverse the fruit array and print the array.
  4. Add code to append "orange" to the fruit array and print the array.
  5. Add code to insert "cherry" before "apple" in the fruit array and print the array.
  6. Add code to sort the fruit array and print the array.
  7. Add code to clear the fruit array and print the array.

Sample Run

> python check_solution.py
original: ['pear', 'banana', 'apple', 'mango']
reversed: ['mango', 'apple', 'banana', 'pear']
append orange: ['mango', 'apple', 'banana', 'pear', 'orange']
insert cherry: ['mango', 'cherry', 'apple', 'banana', 'pear', 'orange']
remove banana: ['mango', 'cherry', 'apple', 'pear', 'orange']
sorted: ['apple', 'cherry', 'mango', 'orange', 'pear']
cleared: []

Testing Procedure

Verify that your program works correctly by following each step in this testing procedure:

  1. Run your program and ensure that your program's output is the same as the sample run output.

Ponder

After you finish your program, examine the code in main and realize that you wrote object oriented code when you used the fruit array and dot notation to call the reverse, append, insert, remove, sort, and clear methods.
