06 Prepare: Writing Functions

Because most useful computer programs are very large, programmers divide their programs into parts. Dividing a program into parts makes it easier to write, debug, and understand the program. A programmer can divide a Python program into modules, classes, and functions. In this lesson, you will learn how to write your own functions.

Here are the Python programming concepts and topics that you should learn this week.

  1. A function is a group of statements (computer commands) that together perform one task.
