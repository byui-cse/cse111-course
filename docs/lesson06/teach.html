06 Team Activity: Writing Functions

Problem Statement

Many programs need to read and write text files. Often that text will be in languages other than English. Early electronic computers were able to process and store English characters only. In 1987, several individuals began investigating the possibility of a universal character set that would enable computers to store and transmit characters from all human languages. Their work led to the international standard known as Unicode. The Unicode standard assigns a unique number to every character or almost every character written by humans throughout the history of the world.

During this exercise you will learn to open a text file for writing and write text to it, including text in languages other than English. You will also learn about the optional parameters of the open function.

Assignment

Write a Python program named text_file.py with three functions named write_text_file, read_text_file, and main as follows:

  1. write_text_file
    1. Has two parameters named filename and text
    2. Opens the text file named filename for writing with the utf-8 encoding.
    3. Writes all the text to the file.
  2. read_text_file
    1. Has one parameter named filename.
    2. Opens a text file for reading with the utf-8 encoding.
    3. Reads all the text in the file into a string.
    4. Return the string.
  3. main
    1. Has no parameters.
    2. Calls write_text_file to write the following string to a file named "forgive.txt". 
      verse = """For if ye forgive men their trespasses,
your Heavenly Father will also forgive you.
ἐὰν γὰρ ἀφῆτε τοῖς ἀνθρώποις τὰ παραπτώματα αὐτῶν,
ἀφήσει καὶ ὑμῖν ὁ πατὴρ ὑμῶν ὁ οὐράνιος.
𐐙𐐫𐑉, 𐐮𐑁 𐐷 𐑁𐐲𐑉𐑀𐐮𐑂 𐑋𐐯𐑌 𐑄𐐯𐑉 𐐻𐑉𐐯𐑅𐐹𐐰𐑅𐐮𐑆,
𐐷𐐫𐑉 𐐸𐐯𐑂𐐲𐑌𐑊𐐨 𐐙𐐱𐑄𐐲𐑉 𐐶𐐮𐑊 𐐫𐑊𐑅𐐬 𐑁𐐲𐑉𐑀𐐮𐑂 𐐷𐐭."""
    3. Calls read_text_file to read the contents of forgive.txt.
    4. Prints the text returned from read_text_file.

    Notice that the string includes characters from English, Greek, and the Deseret alphabet. It is important that the test string includes characters from other languages besides English so that you can verify that your write_text_file and read_text_file functions are working correctly.

  4. At the bottom of your program, write a line of code to call the main function. Then run your program and verify that your program correctly writes and reads the verse string from above. All the characters in forgive.txt should look the same as they look in this web page.

Core Requirements

Your program must include the following functions, each working correctly. Each function must have its own document string (triple quoted string) that explains what the function does.

  1. write_text_file
  2. read_text_file
  3. main

Stretch Challenges

If your team finishes the core requirements in less than an hour, complete one or more of these stretch challenges. Note that the stretch challenges are optional.

  1. Add something or change something in your program that you think would make your program better, easier for the user, more elegant, or more fun. Be creative.
  2. Write comments above the lines of code that were difficult for you to understand.
  3. Add a function named copy_text_file that takes two parameters named source_filename and target_filename. The function must call read_text_file and then write_text_file and thus copy the contents of source_filename into target_filename.

Testing Procedure

Verify that your program works correctly by following each step in this testing procedure:

  1. Run your program and verify that your program correctly wrote the multi-language string to the forgive.txt file. Open the forgive.txt file in VS Code. All the characters in forgive.txt should look the same as they look above in this web page.
