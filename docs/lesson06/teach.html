CSE 111 | Programming with Functions

06 Team Activity: Writing Text to a File

Instructions

Arrange a one hour synchronous meeting with your team for this activity. Online students should coordinate a video-sharing meeting. Campus students will use class time for this meeting. You should prepare for this meeting by completing the preparation material and the individual checkpoint assignment beforehand.

Problem Statement

Many programs need to read and write text to files. Often that text will be in languages other than English. During this exercise you will learn to open a text file and write text to it, including text in languages other than English. You will also learn about the optional arguments to the open function.

Assignment

Write a Python program that opens a file named forgive.txt and writes these three strings to that file.

english_matt_6_14 = "For if ye forgive men their trespasses, your heavenly Father will also forgive you\n"
greek_matt_6_14 = "ἐὰν γὰρ ἀφῆτε τοῖς ἀνθρώποις τὰ παραπτώματα αὐτῶν, ἀφήσει καὶ ὑμῖν ὁ πατὴρ ὑμῶν ὁ οὐράνιος\n"
deseret_3_nephi_13_14 = "𐐙𐐫𐑉, 𐐮𐑁 𐐷 𐑁𐐲𐑉𐑀𐐮𐑂 𐑋𐐯𐑌 𐑄𐐯𐑉 𐐻𐑉𐐯𐑅𐐹𐐰𐑅𐐮𐑆, 𐐷𐐫𐑉 𐐸𐐯𐑂𐐲𐑌𐑊𐐨 𐐙𐐱𐑄𐐲𐑉 𐐶𐐮𐑊 𐐫𐑊𐑅𐐬 𐑁𐐲𐑉𐑀𐐮𐑂 𐐷𐐭\n"

Core Requirements

  1. Your program must correctly open a text file named forgive.txt and save the English version of Matthew 6:14 into it.
  2. Your program must correctly save the Greek version of Matthew 6:14 into forgive.txt.
  3. Your program must correctly save the Deseret version of 3 Nephi 13:14 into forgive.txt.

Stretch Challenges

If your team finishes the core requirements in less than an hour, complete one or more of these stretch challenges. Note that the stretch challenges are optional.

  1. Add something or change something in your program that you think would make your program better, easier for the user, more elegant, or more fun. Be creative.
  2. Add code to your program so that it writes the text of 3 Nephi 13:14 in another language besides English, Greek, and Deseret.
  3. Add a feature to your program that allows the user to choose which language will be written to the file.
  4. Write comments above the lines of code that were difficult for you to understand.

Testing Procedure

Verify that your program works correctly by following each step in this testing procedure:

  1. Run your program. It should complete without errors and should create a text file named forgive.txt.
  2. Open the forgive.txt file in VS Code. All the characters in forgive.txt should look the same as they look above in this web page.
