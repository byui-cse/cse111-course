CSE 111 | Programming with Functions

06 Checkpoint: Default Parameter Values and Optional Arguments

Overview

Demonstrate your understanding of writing a function with an optional parameter and calling that function by completing the following individual checkpoint assignment.

Helpful Documentation

The Python random module contains functions to generate pseudo random numbers. You can read about the uniform function in the random module section of the official online Python documentation.

The built-in Python round function rounds a number to a specified number of digits after the decimal place. You can read about the round function in the built-in functions section of the official online Python documentation.

Assignment

Write a program with two functions named main and append_random_numbers as follows:

  1. main()
    1. has no parameters
    2. creates a list named randnums like this: 
      randnums = [16.2, 75.1, 52.3]
    3. prints randnums
    4. calls the append_random_numbers function once with only one argument to add one number to randnums
    5. prints randnums
    6. calls the append_random_numbers function once more with two arguments to add three numbers to randnums
    7. prints randnums
  2. append_random_numbers()
    1. has two parameters: a list named ls and an integer named n
    2. the parameter n has a default value of 1
    3. computes n pseudo random numbers by calling the random.uniform function.
    4. rounds the n pseudo random numbers to one digit after the decimal.
    5. appends the n pseudo random numbers onto the end of the ls list
  3. Calls main() to begin your program executing

Sample Run

> python check_solution.py
randnums [16.2, 75.1, 52.3]
randnums [16.2, 75.1, 52.3, 39.1]
randnums [16.2, 75.1, 52.3, 39.1, 11.7, 98.9, 23.7]

Note that the output of your program will be slightly different than the output shown above in the sample run. Specifically, the last four random numbers will be different.

Testing Procedure

Verify that your program works correctly by following each step in this testing procedure:

  1. Run your program and ensure that your program's output is similar to the sample run output. Because your program is appending random numbers to the randnums list, your program's output will show some different numbers than the sample run.
