09 Checkpoint: Water Meter Readings

Overview

Complete this individual checkpoint assignment to demonstrate that you can use pandas to read a CSV file and print a summary of the data.

Problem Statement

In many countries of the world, including the USA, drinking water is delivered to houses through underground pipes. Very often this water must be filtered and purified before it is put into the pipes. Of course, filtering and purification costs money, and most cities charge households according to the amount of water that the household uses. In order to know how much water a household uses, the city inserts a device called a water meter in the pipe immediately before each house. The water meter measures how much water passes by it in the pipe. An employee of the city comes frequently to read and record the amount of water used as shown on the meter.

Employees of the City of Rexburg, Idaho, where BYU-Idaho is located, read the water meters about once each month. For the checkpoints and assignments of this lesson and the next, you will write Python code that reads and processes a CSV file named water.csv that contains water meter readings for five years from the City of Rexburg.

Assignment

Write a Python program with a main function that uses pandas to read a CSV file into a DataFrame and to manipulate and print the data. Do the following:

  1. Open VS Code and open a Python file (.py) so that VS Code enables the Python extension.
  2. Open the VS Code terminal frame: on the menu bar for VS Code, click "Terminal" and then "New Terminal."
  3. Upgrade pip, setuptools, and wheel by executing this command: 
    python -m pip install --user --upgrade pip setuptools wheel
  4. Install the numpy, matplotlib, and pandas modules: 
    python -m pip install --user numpy matplotlib pandas
  5. Download and save this water.zip file.
  6. Unzip the file (extract all) into the same folder where you will save your Python program. This will unzip the file water.csv This file contains water meter readings for several years from the City of Rexburg, Idaho.
  7. Open the unzipped water.csv file in VS Code and examine it. Notice that each row in the file contains seven columns which are named: meterNumber, meterSize, readDate, numberOfDays, usage, accountType, and numberOfDwellings. Notice also that this file is large, containing 246,832 lines. In Rexburg, each water meter is read about once a month. Each line in the water.csv file, except for the first line, holds the data from reading one water meter one time.
  8. Write a Python program with a main function that does the following in this order:
    1. Uses pandas.read_csv to read the water.csv file into a DataFrame.
    2. Prints the DataFrame's .dtype attribute.
    3. Calls the .describe() function and prints the summary returned from .describe().
    4. Calls the print() function to print the DataFrame.
    5. Filters the DataFrame to include residences only.
    6. Calls the print() function to print the filtered DataFrame.
