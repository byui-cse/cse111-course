09 Checkpoint: Data Analysis

Overview

Complete this individual checkpoint assignment to demonstrate your ability to use pandas to read a CSV file and to print a summary of the data.

Helpful Documentation

Assignment

Write a Python program with a main function that uses pandas to read a CSV file into a DataFrame and to print a summary of the data. Do the following:

  1. Open VS Code and open its terminal window: on the menu bar for VS Code, click "Terminal" and then "New Terminal".
  2. Upgrade pip, setuptools, and wheel by executing this command: 
    python -m pip install --user --upgrade pip setuptools wheel
  3. Install the numpy, matplotlib, and pandas modules: 
    python -m pip install --user numpy matplotlib pandas
  4. Download and save this water.zip file.
  5. Unzip the file (extract all) into the same folder where you will save your Python program. This will unzip the file water.csv This file contains water meter readings for several years from the City of Rexburg, Idaho.
  6. Open the unzipped water.csv file in VS Code and examine it. Notice that each row in the file contains six columns which are named: meterNumber, readDate, numberOfDays, usage, accountType, and numberOfDwellings. Notice also that this file is large, containing 226,894 lines. In Rexburg, each water meter is read about once a month. Each line in the water.csv file, except for the first line, holds the data from reading one water meter one time.
  7. Write a Python program with a main function that does the following in this order:
    1. Uses pandas.read_csv to read the water.csv file into a DataFrame.
    2. Calls the .describe() function and prints the summary returned from .describe()
    3. Calls the print function to print the data.
