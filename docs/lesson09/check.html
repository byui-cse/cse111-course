09 Checkpoint: Data Analysis

Overview

Complete this individual checkpoint assignment to demonstrate your ability to use pandas to read a CSV file and retrieve, update, and sort data.

Helpful Documentation

Assignment

Write a Python program with a main function that uses pandas to read a CSV file into a DataFrame and to retrieve, update, sort and print data from the DataFrame. Do the following:

  1. Update setup, wheel, etc
  2. Install numpy
  3. Install matplotlib
  4. Install pandas
  5. Download and save this readings.zip file.
  6. Unzip the file (extract all) into the same folder where you will save your Python program. This will unzip the file readings.csv This file contains water meter readings for several years from the City of Rexburg, Idaho.
  7. Open the unzipped readings.csv file in VS Code and examine it. Notice that each row in the file contains nine columns which are named: meterNumber, priorReadDate, readDate, startRead, endRead, stubSize, numberOfDwellings, accountNumber, and accountType. Notice also that this file is large, containing 279,697 lines. In Rexburg, each water meter is read about once a month. Each line of the readings.csv file, except for the first line, holds the data from reading one water meter one time.
