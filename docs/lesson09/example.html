CSE 111 | Programming with Functions

09 Example: Text Files

Purpose

Prove that you can write a Python program that reads CSV files and creates, populates, and uses a dictionary.

Problem Statement

A local music teacher has a large collection of recorded music. He has a CSV file named all_music.csv that contains information about all of his music. He also has a playlist stored in a text file named evening.txt. He wants to know how long it will take to listen to the songs in the playlist.

The prepare content for this lesson includes a section about using lists and a section about reading text files.

Assignment

Write a program that does the following:

  1. Reads the data in the all_music.csv file and stores it in a dictionary named all_music with the song title as the key.
  2. Reads the titles from the evening.txt playlist.
  3. For each title in the playlist does the following:
    1. Finds the title in the all_music dictionary.
    2. Gets the length in seconds of the song.
    3. Prints the song title and length in seconds.
    4. Adds the length to a total.
  4. Prints the total length of the playlist in minutes.
