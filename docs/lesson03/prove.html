03 Prove: Milestone

Overview

Prove that you can write a Python program that calls functions.

Problem Statement

The weather affects people's lives for good and bad every day. Because of the impact of the weather on people's lives, the United States Federal Government sponsors the U.S. National Weather Service which forecasts the weather throughout the entire United States every day. These weather forecasts are available for free as a web service on the internet. Write a Python program that

  1. allows a user to select a city from a list of cities
  2. retrieves weather forecast for the next seven days and nights from the U.S. National Weather Service
  3. writes the forecast to an HTML file that a user can view in a browser

During this assignment (lesson 3), you will write half of this Python program. During this assignment, your program will not retrieve the forecast from the U.S. Weather Service but will instead use two sample forecasts and will write those forecasts to an HTML file. During lesson 4, you will add the code to prompt the user for their desired city and to retrieve a forecast from the U.S. Weather Service.

Helpful Documentation

The Python weather_html module was written at BYU-Idaho and accepts U.S. National Weather Service forecasts and writes HTML to a file. To prepare for this assignment, read about the weather_html module and its functions.

Assignment

Do the following:

  1. Download both of these files: weather_html.py and test_periods.py and save them into the same folder where you will save your Python program.
  2. Open the weather_html.py file and examine it. Notice that this file is the weather_html module and contains the four functions: create_report, start_new_row, add_period, write_report that are described in the module's documentation.
  3. Open the test_periods.py file and notice that it contains two lists: periods_that_start_with_a_day and periods_that_start_with_a_night. You will use these two lists to test your program during this assignment.
  4. Use this code to begin your Python program 
    import test_periods
import weather_html

city_name = "Rexburg, Idaho"
periods = test_periods.periods_that_start_with_a_day
# periods = test_periods.periods_that_start_with_a_night

  5. Finish your program by writing code to call the four functions from the weather_html module as necessary to produce an HTML document in a five day vertical format. Your program should also include if statements and loops as necessary. Here are two sample reports in a five day vertical format.

    1. forecast5day.html
    2. forecast5night.html

    The first report is for a forecast that was retrieved early in a morning and therefore begins with a day period. The second report is for a forecast that was retrieved in an afternoon and therefore begins with a night period. The reports that your program creates should look like the sample reports.

Testing Procedure

Verify that your program works correctly by following each step in this testing procedure:

  1. Run your program and double click on the HTML file that you program produces. Double clicking the HTML file should open it in a browser. Look at the document in the browser and verify that it looks like the forecast5day.html document.
  2. Change your program to use the list periods_that_start_with_a_night and then run your program again. Double click on the HTML file and verify that the document changed and that it looks like the forecast5night.html document.
