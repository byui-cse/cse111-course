CSE 111 | Programming with Functions

03 Prove Milestone: Weather Forecast

Overview

Prove that you can write a Python program that calls functions.

Problem Statement

The weather affects people's lives for good and bad every day. Because of the impact of the weather on people's lives, the United States Federal Government sponsors the U.S. National Weather Service which forecasts the weather throughout the entire United States every day. These weather forecasts are available for free as a web service on the internet. Write a Python program to do the following:

  1. Allow a user to select a city from a list of cities.
  2. Retrieve weather forecast for the next seven days and nights from the U.S. National Weather Service.
  3. Write the forecast to an HTML file that a user can view in a browser.

During this assignment (lesson 3), you will write half of this Python program. During this assignment, your program will not retrieve the forecast from the U.S. Weather Service, but it will instead use two sample forecasts and write those forecasts to an HTML file. During lesson 4, you will add the code to prompt the user for their desired city and to retrieve a forecast from the U.S. Weather Service.

Helpful Documentation

The Python weather_html module was written at BYU-Idaho and accepts U.S. National Weather Service forecasts and writes HTML to a file. To prepare for this assignment, read about the weather_html module and its functions.

Assignment

Do the following:

  1. Download both of these files: weather_html.py and test_periods.py and save them into the same folder where you will save your Python program.
  2. Open the weather_html.py file and examine it. Notice that this file is the weather_html module and contains the four functions: create_report, start_new_row, add_period, write_report that are described in the module's documentation.
  3. Open the test_periods.py file and notice that it contains two lists: periods_that_start_with_a_day and periods_that_start_with_a_night. You will use these two lists to test your program during this assignment.
  4. Use this code to begin your Python program: 
    import test_periods
import weather_html

city_name = "Rexburg, Idaho"
periods = test_periods.periods_that_start_with_a_day
# periods = test_periods.periods_that_start_with_a_night
  5. Finish your program by writing code to call the four functions from the weather_html module as necessary to produce an HTML document in a five day vertical format. Your program should also include if statements as necessary and a loop to iterate through the periods. See the "Testing Procedure" section below for sample reports.

Testing Procedure

Verify that your program works correctly by following each step in this testing procedure:

  1. Run your program and double click on the HTML file that you program produces. Double clicking the HTML file should open it in a browser. Look in the browser at the document that your program produced and verify that it looks like this sample forecast5day.html document. This sample document came from the periods_that_start_with_a_day list and therefore begins with a day period.
  2. Change your program to use the periods_that_start_with_a_night list and then run your program again. Double click on the HTML file and verify that the document that your program produced now looks like this sample forecast5night.html document. This sample document came from the periods_that_start_with_a_night list and therefore begins with a night period.
