CSE 111 | Programming with Functions

03 Prove: Assignment Milestone

Overview

Write a Python program that calls built-in functions, standard functions, and functions in a third party library.

Project Description

In the spring of 2019, several BYU-Idaho students created a small Android app to help people vocabulary words in a foreign language. This app stores the vocabulary words in a centralized firebase database. Your assignment during lessons 3 and 4 is to write a Python program that runs on your computer and enables a user to do the following:

  1. select a language
  2. list all of the words stored in the database for that language
  3. find a word and its translation in the database
  4. add a new word and its translation to the database
