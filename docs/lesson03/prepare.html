03 Prepare: Lists and Dictionaries

Concepts

Here are the Python programming concepts and topics that you should learn during this lesson:

Lists

Within a Python program, we can store many elements in a list. Lists are mutable, meaning they can be changed after they are created. Each element in a list is stored at a unique index. An index is always an iteger, and the first index in a Python list is always zero (0). The following diagram shows a list that contains five words. The diagram shows both the elements and the indexes of the list. Notice that each index is a unique integer, and that the first index is zero.

elements "yellow" "red" "green" "yellow" "blue"
indexes [0] [1] [2] [3] [4]
← list length is 5 →

We can create a list by using square brackets ([ and ]). We can add an item to a list using the insert and append methods. We can find an item in a list using index. We can determine the number of items in a list using the len function. We can retrieve an item from a list and replace an item in a list using square brackets ([ and ]) and an index. We can remove an item from a list using del, pop, and remove. The following code example shows how to create a list and add, find, and remove items from a list.

# Create a list that contains five words.
colors = ["yellow", "red", "green", "yellow", "blue"]

# Print the entire colors list.
print(colors)

# Print the element that is stored at index 2 in the colors list.
print(colors[2])


# Create an empty list that will hold fabric names.
fabrics = []

# Add three elements at the end of the fabrics list.
fabrics.append("velvet")
fabrics.append("denim")
fabrics.append("gingham")

# Insert an element at the beginning of the fabrics list.
fabrics.insert(0, "chiffon")

# Get the index where velvet is stored in the fabrics list.
i = fabrics.index("velvet")

# Replace velvet with taffeta.
fabrics[i] = "taffeta"

# Remove the last element from the fabrics list.
fabrics.pop()

# Remove taffeta from the fabrics list.
fabrics.remove("taffeta")

# Get the length of the fabrics list and print it.
n = len(fabrics)
print(f"The fabrics list contains {n} elements.")

Dictionaries

Within a Python program, we can store many items in a dictionary. Dictionaries are mutable, meaning they can be changed after they are created. In a dictionary, items are stored as key value pairs. All keys within a dictionary must be unique. In other words, no key can appear more than once in a dictionary. Values within a dictionary do not have to be unique. students = {

Line 12 in the previous code example, adds an item to the dictionary. To add an item to an existing dictionary, write code that follows this template:

Line 26 in the previous code example, looks up a key and retrieves its corresponding value from a dictionary. To look up a key and retrieve its corresponding value, write code that follows this template: 

value = dictionary_name[key]
Notice that line 26 follows this template.

Converting between Lists and Dictionaries

It is possible to convert two lists into a dictionary by using the built-in zip and dict functions. The contents of the first list will become the keys in the dictionary, and the contents of the second list will become the values. This implies that the two lists must have the same length, and the elements in the first list must be unique.

It is also possible to convert a dictionary into two lists by using the keys and values methods and the built-in list function. The following code example starts with two lists, converts them into a dictionary, and then converts the dictionary into two lists.

# Create a list that contains five student numbers.
numbers = ["42-039-4736", "61-315-0160",
        "10-450-1203", "75-421-2310", "07-103-5621"]

# Create a list that contains five student names.
names = ["Clint Huish", "Michelle Davis",
        "Jorge Soares", "Abdul Ali", "Michelle Davis"]

# Convert the numbers list and names list into a dictionary.
student_dict = dict(zip(numbers, names))

# Print the entire student dictionary.
print(student_dict)

# Convert the student dictionary into
# two lists named keys and values.
keys = list(student_dict.keys())
values = list(student_dict.values())

# Print both lists.
print(keys)
print(values)

Text Files

The built-in open() function opens a file for reading or writing. The file.read() method reads characters or bytes from a file. The file.readline() method reads one line of text from a text file. The file.readlines() method reads all the lines of text from a text file and returns the lines in a list. The following code example uses the built-in open function and readlines method to read the contents of a text file into a list.

# Open the plants.txt file for reading.
with open("plants.txt", "rt") as infile

    # Read all the lines of the file into a list
    # named plants. Each line of the file will be
    # stored in its own element in the list.
    plants = infile.readlines()

    # Print the entire list.
    print(plants)

Comma Separated Values

Many computer systems import and export data in CSV files. CSV is an acronym for comma separated values. A CSV file is a text file that contains tabular data where each row is on a separate line and each cell (column) is separated by a comma. The following example is the contents of a CSV file that stores information about some dental offices in Rexburg, Idaho. Notice that the first row of the file contains column headings, the next four rows contain information about four dental offices, and each row contains three columns separated by commas.

Company Name,Address,Phone Number
Eagle Rock Dental Care,556 Trejo Suite C,208-359-2224
Apple Tree Dental,33 Winn Drive Suite 2,208-359-1500
Rockhouse Dentistry,106 E 1st N,208-356-5600
Cornerstone Family Dental,44 S Center Street,208-356-4240

Python has a module named csv that includes functionality to read and write data from CSV files. The following example code shows how to open a CSV file and use the csv module to read and print each row.

import csv

# Open a file named dentists.csv and store a reference
# to the opened file in a variable named infile.
with open("dentists.csv", "rt") as infile:

    # Use the csv module to create a reader
    # object that will read from the opened file.
    reader = csv.reader(infile)

    # The first line of the CSV file contains headings
    # and not information about a dental office, so this
    # statement skips the first line of the CSV file.
    next(reader)

    # Read each row one at a time as a list.
    for row in reader:

        # For the current row, retrieve the
        # values in columns 0, 1, and 2.
        company = row[0]
        address = row[1]
        phone = row[2]

        # Print the values for the user to see.
        print(company, address, phone)

You could use this example code in a program that you write, but of course, you could modify the for loop in the code to do much more than simply print the contents of each row. For example, the next code sample shows how to read the contents of a CSV file into a dictionary.

import csv

# Create an empty dictionary that will
# store the data from the CSV file.
dentists = {}

# Open a file named dentists.csv and store a reference
# to the opened file in a variable named infile.
with open("dentists.csv", "rt") as infile:

    # Use the csv module to create a reader
    # object that will read from the opened file.
    reader = csv.reader(infile)

    # The first line of the CSV file contains headings
    # and not information about a dental office, so this
    # statement skips the first line of the CSV file.
    next(reader)

    # Read each row one at a time as a list.
    for row in reader:

        # For the current row, retrieve the
        # values in columns 0, 1, and 2.
        company = row[0]
        address = row[1]
        phone = row[2]

        # Store the data from the current row into the dentists
        # dictionary. Use the phone number as the key. Create a
        # list with the company name and address as the value.
        dentists[phone] = [company, address]

# Print the entire dentists dictionary.
print(dentists)
