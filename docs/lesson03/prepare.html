CSE 111 | Programming with Functions

03 Prepare: Calling Functions

Because most useful computer programs are very large, programmers divide their programs into parts. Dividing a program into parts makes it easier to write, debug, and understand the program. A programmer can divide a Python program into modules, classes, and functions. In this unit, you will learn how to use existing functions, and in the next unit, you will learn how to write your own functions.

Here are the Python programming concepts and topics that you should learn this week:

  1. A function is a group of statements (computer commands) that together perform one task.
  2. Python includes many built-in functions such as: input, print, open, len, range, int, float, str, abs, round, list, set, and dict. You can read about the Python built-in functions at docs.python.org/3/library/functions.html
  3. Python also includes many standard modules which have more functions, such as the math module which includes the floor, ceil, and sqrt functions and the csv module which includes the reader and writer functions. You can read more about the Python modules at https://docs.python.org/3/py-modindex.html
  4. A programmer (you) can save lots of time by using existing functions. A programmer uses a function by calling it (also known as invoking it). To call (or invoke) a function means to write code that causes the computer to execute the code that is inside of that function. During CSE 110, you often wrote code that called the input and print functions. For example: 
    name = input("Please enter your name")
print(f"Hello {name}")
  5. To correctly call a function you must know the following:
    1. the name of the module in which the function is located
    2. the name of the function
    3. the parameters that the function takes
    4. what the function does

    All of this information is normally available in online documentation. For example, from the online Python documentation about the math module, we read this about the sqrt function:

    math.sqrt(x)
            Return the square root of x.

    From this short description, we know that

    1. the name of its containing module is math
    2. the name of the function is sqrt
    3. the function takes one parameter (x)
    4. the function computes and returns the square root of a number

    You can read about

    • the built-in functions at docs.python.org/3/library/functions.html,
    • the math module at docs.python.org/3/library/math.html
    • the csv module at docs.python.org/3/library/csv.html.

  6. A parameter is a piece of data that a function needs in order to complete its task. In the online documentation for the math module, we see that the math.sqrt function has one parameter named x which is the number that math.sqrt will compute the square root of.

    math.sqrt(x)
            Return the square root of x.

    In order for the math.sqrt function to do its job (compute the square root), a programmer must write code to pass a number through the parameter x into into the math.sqrt function. Here is a simple example of passing a number to the math.sqrt function:

    r = math.sqrt(71)

    In the above example, 71 will be passed through the parameter x into the math.sqrt function. The math.sqrt function will compute the square root of 71 and return the computed value which will then be stored in the variable r.

    Here is another example that gets a number from the user and then computes and prints the square root of that number. Notice that this example calls four functions: input, float, math.sqrt, and print. The code in this example first calls the input function to get input from the user, then calls the float function to convert that input to a number, then calls the math.sqrt function to compute the square root, and finally calls the print function to display the result for the user to see.

    num = float(input("Please enter a number: "))
rt = math.sqrt(num)
print(rt)

    Here is an example that calls the built-in input and open functions to open a text file and then calls the reader function that is in the csv module:

    filename = input("Please enter a filename: ")
with open(filename, "rt") as infile:
    reader = csv.reader(infile)
