weather_html
Write U.S. Weather Forecasts to an HTML File

The weather_html module writes U.S. National Weather Service forecasts to an HTML file.

To successfully use this module to write an HTML file, your code must first retrieve a weather forecast from the U.S. National Weather Service. The weather service organizes the weather forecast into periods of time. Each period covers about 12 hours and is either a day, an afternoon, or a night.

After your code retrives the periods from the U.S. Weather service, it should call these functions from this module:

  1. weather_html.create_report(filename, locname, options)
    Creates and returns a weather report that will be used for adding rows and periods and then writing the report. create_report must be called once before calling the other functions in this module.
    • param filename: path and name of the HTML file where this module will write the weather forecast.
    • param locname: name of the geographic location (city) where the forecast apply.
    • param options: an optional list of weather forecast data that the user desires to be written in a report.
  2. weather_html.start_new_row(report)
    Begins a new row of periods in a weather report. start_new_row must be called once at the beginning of each desired row of periods.
    • param report: the report to add the new row to
  3. weather_html.add_period(report, period)
    Adds one forecast for a period to the current row in the weather report. add_period must be called once for each period in the current row.
    • param report: the report to add the period to
    • param period: a forecast for a period of time that came from the U.S. National Weather Service
  4. weather_html.write_report(report)
    Writes the weather forecast to an HTML file. write_report must be called once, after your code has added all desired rows and periods.
    • param report: the report that will be written to an HTML file.

Example

The following example code will generate a simple HTML forecast report with one row with two periods in that row. Each period in the report will show only the period name and the temperature.

import weather_html

periods = // get a forecast from the U.S. Weather service

# Create a report that will show the period name and temperature.
report = weather_html.create_report("forecast.html", "Rexburg, Idaho",
    [weather_html.PERIOD_NAME, weather_html.TEMPERATURE])

# Start a new row in the report.
weather_html.start_new_row(report)

# Add a period from the forecast to the current row.
weather_html.add_period(report, periods[0])

# Add a second period from the forecast to the current row.
weather_html.add_period(report, periods[1])

# Finish the report and write it to the HTML file.
weather_html.write_report(report)
