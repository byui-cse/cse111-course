CSE 111 | Programming with Functions

03 Checkpoint: Calling Functions

Overview

Demonstrate your understanding of calling Python built-in functions by completing the following individual checkpoint assignment.

Helpful Documentation

You can read about the Python built-in functions at docs.python.org/3/library/functions.html

Assignment

Do the following:

  1. In your text editor, open a new file for a Python program.
  2. Copy and paste this Python code that creates a list into your program. 
    heights = [
    1.72, 1.50, 1.96, 2.01, 1.75,
    2.11, 1.60, 1.65, 2.05, 1.50,
    1.80, 1.83, 2.05, 1.79, 1.84
]
  3. Write Python code to call each of these built-in functions and pass the list of heights to each function. Store the value returned from each function in a different variable.
    1. len
    2. min
    3. max
    4. sum
  4. Write code to call the round function to round the value returned from the sum function to two decimal places and store the result in a variable.
  5. Write code to print the value returned from each of the functions.

Sample Run

> python check_solution.py
len: 15
min: 1.5
max: 2.11
sum: 27.16

Testing Procedure

Verify that your program works correctly by following each step in this testing procedure:

  1. Run your program using the input shown in the sample run section above. Ensure that your program's output matches the sample run output.
