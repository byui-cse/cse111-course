Reading Files

You must do the following to setup VS Code so that your program can read from a text file:

  1. Download the text file (.txt, .csv, etc.) and save it in the same folder where you will save your Python program.
  2. Open VS Code.
  3. Using VS Code, open the folder where you saved the text file file by doing the following:
    • On a computer running the Windows operating system:
      1. Click the "File" menu, then "Open Folder..."
      2. Find and select the folder where you saved the text file.
      3. Click the "Select Folder" button.
    • On a computer running the Mac OSX operating system:
      1. Click the "File" menu, then "Open..."
      2. Find and select the folder where you saved the text file.
      3. Click the "Open" button.
