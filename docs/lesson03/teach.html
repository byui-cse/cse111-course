03 Team Activity: Randomly Chosen Quotes

Instructions

Arrange a one hour synchronous meeting with your team for this activity. Online students should coordinate a video-sharing meeting. Campus students will use class time for this meeting. You should prepare for this meeting by completing the preparation material and the individual checkpoint assignment beforehand.

Problem Statement

When working on a long task or completing a long term goal, people sometimes find it difficult to stay motivated. Some people find that inspiring quotes from others who accomplished similar tasks help them to stay motivated. During this activity, your team will write a Python program that prints a quote choosen randomly from a file of quotes.

Helpful Documentation

The Python random module contains functions to generate pseudo random numbers and to make pseudo random choices. Read about the random module in one of these articles:

  • Lesson about random at Real Python
  • Official Python documentation about the random module

The Python textwrap module contains functions to format plain text. The module formats text by adding, removing, and moving line breaks in the input text. Read about the textwrap module in this article:

  • Official Python documentation about the textwrap module

Instructions

Download this file: quotes.txt and save it in the same folder where you will save your Python program.

Write a Python program that reads the contents of the quotes.txt file into a list and then randomly chooses one of the quotes and prints it for the user to see. Include these two lines of code at the top of your program to import the functions in the random and textwrap modules:

import random
import textwrap

Core Requirements

  1. Your program must read the quotes from the quotes.txt file into a list and randomly choose one of the quotes from the list.
  2. Your program must wrap the words in the quote onto multiple lines if the quote is longer than 70 characters and then print the wrapped quote for the user to see.
  3. Your program must call the following functions:
    • open - to open the quotes.txt file.
    • file.readlines - to read all the lines in the file as a list
    • random.choice - to randomly choose one of the quotes
    • textwrap.wrap - to wrap the words of the chosen quote to 70 characters or fewer per line
    • print - to print the wrapped lines of the quote

Stretch Challenges

If your team finishes the core requirements in less than an hour, complete one or more of these stretch challenges. Note that the stretch challenges are optional.

  1. Add something or change something in your program that you think would make your program better, easier for the user, more elegant, or more fun. Be creative.
  2. Write comments above the lines of code that were difficult for you to understand.
  3. Change the call to the textwrap.wrap function to wrap the words in the quote to 50 characters instead of 70.
  4. Allow the user to enter a positive integer to choose how many quotes are chosen and printed.

Testing Procedure

Verify that your program works correctly by following each step in this testing procedure:

  1. Run your program and verify that it outputs a quote (or multiple quotes if you completed stretch challenge #4).
  2. Run your program again and verify that it outputs a different quote.
  3. If the quotes that you program printed are short, run your program again until it prints a long quote that wraps onto two or more lines. Verify that each line contains no more than 70 characters.
